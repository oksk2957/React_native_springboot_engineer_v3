## 2. 발견된 문제점

### 2.1 ❌ 치명적: StatisticsController의 Google ID 토큰 처리 누락
**증거**: `StatisticsController.java:29-34`
```java
String username = jwtTokenProvider.getUsername(token);
User user = userService.getUserByUsername(username);
```

**문제**:
- Google ID 토큰(ES256)의 경우 `getUsername()`이 `"google:" + googleId` 반환
- `getUserByUsername("google:...")` 호출로 사용자 조회 실패
- 결과적으로 `userId = null` → 통계 계산 생략

### 2.2 ⚠️ 통계 데이터 구조 불일치 가능성
**증거**: `StatisticsService.java:34-127`
```java
if (userId != null) {
    // 통계 계산 로직
}
```

**문제**:
- `userId`가 null이면 모든 통계가 0으로 반환
- 프론트엔드에서 stats가 null로 표시됨

### 2.3 ⚠️ 통계 테이블 활용도 낮음
**증거**: `StatisticsService.java:132-205`
```java
// getOverallStatisticsFromStatistics() 메소드 존재하나 사용되지 않음
```

**문제**:
- `statistics` 테이블과 `user_answer` 테이블이 혼재
- 통계 계산 시 일관성 없는 데이터 소스 사용

## 4. 예상 수정 코드

### 4.1 StatisticsController.java 수정
```java
@GetMapping
public ResponseEntity<Map<String, Object>> getStatistics(@RequestHeader(value = "Authorization", required = false) String authHeader) {
    if (authHeader == null || !authHeader.startsWith("Bearer ")) {
        return ResponseEntity.status(401).body(null);
    }
    
    try {
        String token = authHeader.replace("Bearer ", "");
        User user;
        if (jwtTokenProvider.isGoogleIdToken(token)) {
            String googleId = jwtTokenProvider.getGoogleIdFromToken(token);
            user = userService.getUserByGoogleId(googleId);
        } else {
            String username = jwtTokenProvider.getUsername(token);
            user = userService.getUserByUsername(username);
        }
        
        return ResponseEntity.ok(statisticsService.getOverallStatistics(user.getId()));
    } catch (Exception e) {
        return ResponseEntity.status(401).body(null);
    }
}
```

### 4.2 getSubjectiveCount 메소드도 동일하게 수정
```java
@GetMapping("/subjective-count")
public ResponseEntity<Map<String, Long>> getSubjectiveCount(@RequestHeader(value = "Authorization", required = false) String authHeader) {
    Map<String, Long> response = new HashMap<>();
    Long userId = null;
    
    if (authHeader != null && authHeader.startsWith("Bearer ")) {
        try {
            String token = authHeader.replace("Bearer ", "");
            User user;
            if (jwtTokenProvider.isGoogleIdToken(token)) {
                String googleId = jwtTokenProvider.getGoogleIdFromToken(token);
                user = userService.getUserByGoogleId(googleId);
                userId = user.getId();
            } else {
                String username = jwtTokenProvider.getUsername(token);
                user = userService.getUserByUsername(username);
                userId = user.getId();
            }
        } catch (Exception e) {
            // 토큰이 유효하지 않아도 전체 개수는 반환하도록 예외 처리
        }
    }
    
    response.put("count", statisticsService.getSubjectiveRemainingCount(userId));
    return ResponseEntity.ok(response);
}
```
