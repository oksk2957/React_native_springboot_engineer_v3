# 정보처리기사 소프트웨어분류 - 애드웨어(Adware) 주관식 단답형 문제

> **정답 키워드:** 애드웨어 (Adware)
> **문제 유형:** 주관식 단답형 (빈칸 채우기 + 직접 질문 + 비교 설명 혼합)

---

## 문제 1 (빈칸 채우기 유형)

**질문:** 소프트웨어 유형 중, 사용자의 동의 없이 또는 사용자가 인지하지 못하는 사이에 컴퓨터에 설치되어 지속적으로 광고(advertisement) 팝업창이나 배너를 표시하는 소프트웨어를 **[     ]** 라고 한다. 이 소프트웨어는 무료 프로그램에 번들로 포함되어 배포되는 경우가 많으며, 사용자의 인터넷 사용 패턴을 분석하여 타겟 광고를 노출하기도 한다.

**정답:** 애드웨어 (Adware)

**개발 생각:** 빈칸 채우기 유형은 정보처리기사 실기 시험에서 가장 빈출되는 출제 형식이다. 문제 설명에 "광고(advertisement) 팝업창", "배너", "무료 프로그램에 번들로 포함", "사용자 동의 없이 설치"라는 키워드가 포함되어 있으면, 정답은 애드웨어로 유도된다. 애드웨어의 어원인 'ad(광고)' + 'ware(소프트웨어)'의 직관적 메타포가 결정적 구분자이다. 스파이웨어는 "사용자 정보 몰래 수집", 랜섬웨어는 "데이터 암호화 후 몸값 요구", 트로이 목마는 "위장된 악성코드"가 키워드이므로, "광고 팝업창 반복 노출"이라는 표현이 애드웨어의 고유 구분자이다. 프롬프트 작성 시 이 키워드를 명시하면 AI가 정답을 애드웨어로 정확히 유도할 수 있다.

---

## 문제 2 (직접 질문 유형)

**질문:** 사용자가 무료 소프트웨어를 다운로드하여 설치할 때, 사용자의 명시적 동의 없이 함께 설치되어 지속적으로 광고 팝업창이나 배너를 표시하는 소프트웨어의 이름은 무엇인가?

**정답:** 애드웨어 (Adware)

**개발 생각:** 직접 질문 유형은 특정 설치 시나리오(무료 소프트웨어 설치 시 번들 설치)와 결과(광고 팝업창·배너 표시)를 연결하여 정답을 유도한다. "무료 소프트웨어 설치 시 함께 설치", "명시적 동의 없이", "광고 팝업창·배너 표시"라는 구체적 기술 키워드가 애드웨어를 직접 지칭한다. 프리웨어는 무료 제공이 핵심이고, 스파이웨어는 정보 수집이 핵심이므로, "광고 노출을 목적으로 한 번들 설치"라는 설치 행위의 특성이 애드웨어 정답 유도에 효과적이다. 프롬프트에서 "무료 소프트웨어 번들을 통한 광고 노출 소프트웨어"라고 지정하면 AI가 애드웨어를 정답으로 생성한다.

---

## 문제 3 (비교 설명 유형)

**질문:** 소프트웨어 유형 중, **프리웨어(Freeware)**는 사용자에게 무료로 제공되는 소프트웨어이고, **스파이웨어(Spyware)**는 사용자의 인터넷 사용 패턴이나 개인정보를 몰래 수집하는 소프트웨어이다. 반면, **[     ]** 은(는) 사용자의 동의 없이 설치되어 지속적으로 광고(advertisement) 팝업창이나 배너를 표시하는 소프트웨어로, 주로 무료 프로그램에 번들로 포함되어 배포된다. 이 용어의 어원은 'advertisement(광고)'의 약자 'ad'에서 유래하였다.

**정답:** 애드웨어 (Adware)

**개발 생각:** 비교 설명 유형은 정답 항목과 대조 항목(프리웨어, 스파이웨어)을 동시에 제시하여, 대조 항목의 특성(무료 제공, 사용자 정보 몰래 수집)과 정답 항목의 특성(광고 팝업창·배너 표시, 번들 설치)을 명확히 구분한다. 이 유형은 정보처리기사 시험에서 소프트웨어 분류 용어들의 차이점을 묻는 변형으로 자주 출제된다. 특히 "어원에 'ad(광고)'가 포함"이라는 메타포적 구분자를 추가하면, 애드웨어와 스파이웨어(Spyware)의 기능적 차이를 명확히 할 수 있다. 프롬프트 작성 시 "프리웨어·스파이웨어와 비교하여 [빈칸]의 특징을 설명하는 문제, 정답은 애드웨어, 어원 구분자 포함"이라고 지정하면, AI가 대조 항목 설명을 배치하고 빈칸에 애드웨어를 정답으로 유도하는 문제를 정확히 생성한다.

---

## AI 프롬프트 작성 가이드

애드웨어를 정답으로 하는 주관식 단답형 문제를 AI에게 생성하도록 요청할 때, 다음 프롬프트 형식을 사용한다:

```
정보처리기사 실기 시험용 주관식 단답형 문제 3개를 생성하라.
- 정답: 애드웨어 (Adware)
- 유형: 빈칸 채우기와 직접 질문과 비교 설명을 혼합
- 각 문제에 정답과 개발 생각(해설)을 포함
- 애드웨어의 결정적 키워드(광고 팝업창, 배너, 무료 프로그램 번들, 사용자 동의 없이 설치, ad 어원)를 문제 설명에 배치하여 정답을 유도
- 프리웨어, 스파이웨어 등 다른 소프트웨어 분류 용어와의 비교 설명을 1개 문제에 포함
- 애드웨어와 스파이웨어의 기능적 유사성으로 인한 혼동 방지를 위해 어원 구분자(ad vs spy)를 명시
```

**프롬프트 작성 핵심 원칙:**
1. **정답 키워드 명시**: 프롬프트에 "정답: 애드웨어"을 직접 지정하면 AI가 정답을 고정하여 문제를 역설계한다.
2. **유형 지정**: 빈칸 채우기 / 직접 질문 / 비교 설명 유형을 명시하면 문제 형식의 다양성이 보장된다.
3. **결정적 구분자 제시**: "광고 팝업창·배너" vs "무료 제공(프리웨어)" vs "사용자 정보 수집(스파이웨어)" 등 각 분류 용어의 고유 키워드를 프롬프트에 포함하면, AI가 오답과 정답을 명확히 구분하는 문제를 생성한다. 특히 애드웨어와 스파이웨어의 혼동 방지를 위해 어원(ad vs spy)을 구분자로 활용한다.
4. **개발 생각 요청**: "개발 생각(해설)을 포함"이라고 지정하면, AI가 문제 설계 의도와 출제 패턴 분석을 함께 제공하여 학습 효율이 향상된다.
