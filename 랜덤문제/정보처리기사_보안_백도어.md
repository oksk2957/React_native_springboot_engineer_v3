# AI 프롬프트 작성 가이드

## 프롬프트 작성 핵심 원칙
1. **정답 키워드 명시**: 프롬프트에 "정답: ( )"을 직접 지정하면 AI가 정답을 고정하여 문제를 역설계한다.
2. **유형 지정**: 1.빈칸 채우기 / 직접 질문 / 비교 설명 유형을 명시하면 문제만 랜덤형식의 질문으로 다양성이 보장되어 작성한다. 정답은 ()으로 고정된다.
3. **결정적 구분자 제시**: ( )에 해당하는 고유 키워드를 프롬프트에 포함하고 AI가 오답과 정답을 명확히 구분하는 문제를 생성한다.

---

# 주관식 단답형 문제 - 백도어 (Backdoor)

## 문제 1
**유형**: 빈칸 채우기

정상적인 인증 과정을 우회하여 시스템에 불법적으로 접근할 수 있게 하는 비밀 통로를 ( )이라고 한다.

**정답**: (백도어)

---

## 문제 2
**유형**: 직접 질문

개발자나 관리자가 시스템 유지보수를 위해 의도적으로 남겨두거나, 공격자가 악의적으로 설치한 정상 인증을 우회하는 비밀 접속 경로를 무엇이라 하는가?

**정답**: (백도어)

---

## 문제 3
**유형**: 비교 설명

시스템의 정상적인 로그인 절차를 거치지 않고 숨겨진 방법으로 접근할 수 있게 하는 보안 취약점이나 악성코드를 ( )이라 하며, 이를 통해 공격자는 탐지되지 않은 채 시스템을 장악할 수 있다.

**정답**: (백도어)
