# 정보처리기사 운영체제 - pwd

## AI 프롬프트 작성 가이드

- **정답 키워드 명시**: 프롬프트에 "정답: ( )"을 직접 지정하면 AI가 정답을 고정하여 문제를 역설계한다.
- **유형 지정**: 1.빈칸 채우기 / 직접 질문 / 비교 설명 유형을 명시하면 문제만 랜덤형식의 질문으로 다양성이 보장되어 작성한다. 2.정답은 ()으로 고정된다.
- **결정적 구분자 제시**: ( )에 해당하는 고유 키워드를 프롬프트에 포함하고 AI가 오답과 정답을 명확히 구분하는 문제를 생성한다.
- **해설 미포함**: 질문과 정답만 작성하며 해설은 작성하지 않는다.

---

## 문제 1

**유형**: 빈칸 채우기

Unix/Linux 계열 운영체제에서 현재 작업 중인 디렉터리의 전체 경로를 출력하는 명령어는 ( )이다.

**정답**: pwd

---

## 문제 2

**유형**: 직접 질문

pwd 명령어의 원래 이름인 "print working directory"에서 "working directory"는 사용자가 현재 위치하고 있는 디렉터리를 의미한다. 이 현재 작업 디렉터리를 변경하는 명령어는 무엇인가?

**정답**: cd

---

## 문제 3

**유형**: 비교 설명

pwd 명령어는 현재 디렉터리의 절대 경로를 출력한다. 반면 상위 디렉터리로 이동할 때 사용하는 명령어와 인자의 조합은 무엇인가?

**정답**: cd ..
