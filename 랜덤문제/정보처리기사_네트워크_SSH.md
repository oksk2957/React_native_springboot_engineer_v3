# 정보처리기사 네트워크 - SSH

## AI 프롬프트 작성 가이드

- **정답 키워드 명시**: 프롬프트에 "정답: ( )"을 직접 지정하면 AI가 정답을 고정하여 문제를 역설계한다.
- **유형 지정**: 1.빈칸 채우기 / 직접 질문 / 비교 설명 유형을 명시하면 문제만 랜덤형식의 질문으로 다양성이 보장되어 작성한다. 2.정답은 ()으로 고정된다.
- **결정적 구분자 제시**: ( )에 해당하는 고유 키워드를 프롬프트에 포함하고 AI가 오답과 정답을 명확히 구분하는 문제를 생성한다.
- **해설 미포함**: 질문과 정답만 작성하며 해설은 작성하지 않는다.

---

## 문제 1

**유형**: 빈칸 채우기

원격 서버에 안전하게 접속하기 위해 암호화된 통신 채널을 제공하는 네트워크 프로토콜을 ( )이라고 한다.

**정답**: SSH

SSH 프로토콜에서 기본적으로 사용하는 TCP 포트 번호는 무엇인가?

## 문제 3

**유형**: 비교 설명

기존 텔넷(Telnet)은 데이터를 평문으로 전송하여 보안에 취약한 반면, SSH는 모든 통신 데이터를 암호화하여 기밀성과 무결성을 보장한다. 이 SSH의 암호화 통신을 가능하게 하는 대표적인 공개키 암호화 알고리즘은 무엇인가?

**정답**: RSA
