# 정보처리기사 프로그래밍 - 배열

## AI 프롬프트 작성 가이드

- **정답 키워드 명시**: 프롬프트에 "정답: ( )"을 직접 지정하면 AI가 정답을 고정하여 문제를 역설계한다.
- **유형 지정**: 1.빈칸 채우기 / 직접 질문 / 비교 설명 유형을 명시하면 문제만 랜덤형식의 질문으로 다양성이 보장되어 작성한다. 2.정답은 ()으로 고정된다.
- **결정적 구분자 제시**: ( )에 해당하는 고유 키워드를 프롬프트에 포함하고 AI가 오답과 정답을 명확히 구분하는 문제를 생성한다.
- **해설 미포함**: 질문과 정답만 작성하며 해설은 작성하지 않는다.

---

## 문제 1

**유형**: 빈칸 채우기

동일한 데이터 타입의 여러 데이터를 연속적인 메모리 공간에 저장하는 자료구조를 ( )이라고 한다.

**정답**: 배열

---

## 문제 2

**유형**: 직접 질문

배열의 각 요소에 접근할 때 사용하는 0부터 시작하는 숫자를 무엇이라 하는가?

**정답**: 인덱스

## 문제 3

**유형**: 비교 설명

배열은 연속적인 메모리 공간에 데이터를 저장하므로 인덱스를 통해 빠른 접근이 가능하지만, 삽입과 삭제 시 데이터를 이동시켜야 하는 단점이 있다. 이처럼 배열의 특정 위치에 데이터를 추가하거나 제거할 때 발생하는 데이터 이동 현상을 무엇이라 하는가?

**정답**: 쉬프트
