# 정보처리기사 네트워크 - 프록시

## AI 프롬프트 작성 가이드

- **정답 키워드 명시**: 프롬프트에 "정답: ( )"을 직접 지정하면 AI가 정답을 고정하여 문제를 역설계한다.
- **유형 지정**: 1.빈칸 채우기 / 직접 질문 / 비교 설명 유형을 명시하면 문제만 랜덤형식의 질문으로 다양성이 보장되어 작성한다. 2.정답은 ()으로 고정된다.
- **결정적 구분자 제시**: ( )에 해당하는 고유 키워드를 프롬프트에 포함하고 AI가 오답과 정답을 명확히 구분하는 문제를 생성한다.
- **해설 미포함**: 질문과 정답만 작성하며 해설은 작성하지 않는다.

---

## 문제 1

**유형**: 빈칸 채우기

클라이언트와 서버 사이에서 중계 역할을 수행하며, 클라이언트의 요청을 대신하여 서버에 전달하고 응답을 받아오는 중간 서버를 ( )라고 한다.

**정답**: 프록시 서버

---

## 문제 2

**유형**: 직접 질문

프록시 서버 중 클라이언트의 요청을 대신하여 외부 서버에 접속하고, 응답을 캐싱하여 동일한 요청 시 빠르게 제공하는 프록시 유형은 무엇인가?

**정답**: 포워드 프록시

---

## 문제 3

**유형**: 비교 설명

기업 내부 네트워크에서 클라이언트가 인터넷에 접속할 때, 프록시 서버를 통해 트래픽을 제어하고 보안 정책을 적용한다. 이처럼 내부 사용자의 외부 접속을 중계하고 필터링하는 프록시 서버의 대표적인 기능은 무엇인가?

**정답**: 캐싱
