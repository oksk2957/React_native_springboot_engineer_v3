# 정보처리기사 네트워크 - ATM

## AI 프롬프트 작성 가이드

- **정답 키워드 명시**: 프롬프트에 "정답: ( )"을 직접 지정하면 AI가 정답을 고정하여 문제를 역설계한다.
- **유형 지정**: 1.빈칸 채우기 / 직접 질문 / 비교 설명 유형을 명시하면 문제만 랜덤형식의 질문으로 다양성이 보장되어 작성한다. 2.정답은 ()으로 고정된다.
- **결정적 구분자 제시**: ( )에 해당하는 고유 키워드를 프롬프트에 포함하고 AI가 오답과 정답을 명확히 구분하는 문제를 생성한다.
- **해설 미포함**: 질문과 정답만 작성하며 해설은 작성하지 않는다.

---

## 문제 1

**유형**: 빈칸 채우기

비동기 전송 모식으로, 고속 데이터 전송을 위해 고정 길이의 셀을 사용하는 통신 기술을 ( )이라고 한다.

**정답**: ATM

---

## 문제 2

**유형**: 직접 질문

ATM에서 데이터를 전송하는 고정 길이의 패킷 단위를 무엇이라고 하는가?

**정답**: 셀

---

## 문제 3

**유형**: 비교 설명

ATM의 셀 구조는 5바이트의 헤더와 48바이트의 페이로드로 구성되는데, 이처럼 고정된 크기의 데이터 단위를 사용하여 회선교환과 패킷교환의 장점을 결합한 방식을 무엇이라고 하는가?

**정답**: ATM
