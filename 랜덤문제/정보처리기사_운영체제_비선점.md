# 정보처리기사 운영체제 - 비선점

## AI 프롬프트 작성 가이드

- **정답 키워드 명시**: 프롬프트에 "정답: ( )"을 직접 지정하면 AI가 정답을 고정하여 문제를 역설계한다.
- **유형 지정**: 1.빈칸 채우기 / 직접 질문 / 비교 설명 유형을 명시하면 문제만 랜덤형식의 질문으로 다양성이 보장되어 작성한다. 2.정답은 ()으로 고정된다.
- **결정적 구분자 제시**: ( )에 해당하는 고유 키워드를 프롬프트에 포함하고 AI가 오답과 정답을 명확히 구분하는 문제를 생성한다.
- **해설 미포함**: 질문과 정답만 작성하며 해설은 작성하지 않는다.

---

## 문제 1

**유형**: 빈칸 채우기

한 프로세스가 CPU를 할당받으면 작업이 완료될 때까지 다른 프로세스가 CPU를 빼앗을 수 없는 방식을 ( ) 스케줄링이라고 한다.

**정답**: 비선점

---

## 문제 2

**유형**: 직접 질문

비선점 스케줄링의 대표적인 예로 FCFS와 SJF가 있다. 프로세스가 자발적으로 CPU를 반납할 때까지 다른 프로세스가 강제로 CPU를 빼앗을 수 없는 방식을 무엇이라고 하는가?

**정답**: 비선점

---

## 문제 3

**유형**: 비교 설명

선점 스케줄링과 달리 프로세스가 CPU를 한 번 할당받으면 완료될 때까지 다른 프로세스가 방해할 수 없는 스케줄링 방식을 무엇이라고 하는가?

**정답**: 비선점
