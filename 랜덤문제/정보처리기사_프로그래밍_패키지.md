# 정보처리기사 프로그래밍 - 패키지

## AI 프롬프트 작성 가이드

- **정답 키워드 명시**: 프롬프트에 "정답: ( )"을 직접 지정하면 AI가 정답을 고정하여 문제를 역설계한다.
- **유형 지정**: 1.빈칸 채우기 / 직접 질문 / 비교 설명 유형을 명시하면 문제만 랜덤형식의 질문으로 다양성이 보장되어 작성한다. 2.정답은 ()으로 고정된다.
- **결정적 구분자 제시**: ( )에 해당하는 고유 키워드를 프롬프트에 포함하고 AI가 오답과 정답을 명확히 구분하는 문제를 생성한다.
- **해설 미포함**: 질문과 정답만 작성하며 해설은 작성하지 않는다.

---

## 문제 1

**유형**: 빈칸 채우기

관련된 클래스와 인터페이스를 논리적으로 그룹화하여 이름 충돌을 방지하고 코드의 재사용성을 높이는 Java의 개념을 ( )이라고 한다.

**정답**: 패키지

## 문제 2

**유형**: 직접 질문

Java에서 패키지를 선언할 때 사용하는 예약어는 무엇인가?

**정답**: package

## 문제 3

**유형**: 비교 설명

Java에서 다른 패키지에 있는 클래스를 사용하기 위해 해당 클래스의 경로를 명시하거나, 소스 파일 상단에 특정 키워드를 사용하여 패키지를 불러오는데, 이 키워드는 무엇인가?

**정답**: import
