# 정보처리기사 네트워크 - ICMP

## AI 프롬프트 작성 가이드

- **정답 키워드 명시**: 프롬프트에 "정답: ( )"을 직접 지정하면 AI가 정답을 고정하여 문제를 역설계한다.
- **유형 지정**: 1.빈칸 채우기 / 직접 질문 / 비교 설명 유형을 명시하면 문제만 랜덤형식의 질문으로 다양성이 보장되어 작성한다. 2.정답은 ()으로 고정된다.
- **결정적 구분자 제시**: ( )에 해당하는 고유 키워드를 프롬프트에 포함하고 AI가 오답과 정답을 명확히 구분하는 문제를 생성한다.
- **해설 미포함**: 질문과 정답만 작성하며 해설은 작성하지 않는다.

---

## 문제 1

**유형**: 빈칸 채우기

IP 네트워크에서 오류 메시지를 전달하거나 진단 정보를 교환하는 데 사용되는 프로토콜을 ( )이라고 한다.

**정답**: ICMP

---

## 문제 2

**유형**: 직접 질문

ICMP 프로토콜을 사용하여 특정 호스트까지의 네트워크 경로를 추적하거나 연결 상태를 확인할 때 사용하는 대표적인 명령어 도구는 무엇인가?

**정답**: ping

---

## 문제 3

**유형**: 비교 설명

TCP와 UDP는 실제 데이터를 전송하는 전송 계층 프로토콜인 반면, ICMP는 네트워크 계층에서 오류 보고와 진단 기능을 수행한다. ICMP 메시지 중 목적지 호스트나 포트에 도달할 수 없을 때 전송되는 메시지 유형은 무엇인가?

**정답**: Destination Unreachable
