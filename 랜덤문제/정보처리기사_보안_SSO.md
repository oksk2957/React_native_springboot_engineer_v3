# 정보처리기사 주관식 단답형 문제 - SSO (Single Sign-On)

## AI 프롬프트 작성 가이드
- **정답 키워드 명시**: 프롬프트에 "정답: ( )"을 직접 지정하면 AI가 정답을 고정하여 문제를 역설계한다.
- **유형 지정**: 1.빈칸 채우기 / 직접 질문 / 비교 설명 유형을 명시하면 문제만 랜덤형식의 질문으로 다양성이 보장되어 작성한다. 정답은 ()으로 고정된다.
- **결정적 구분자 제시**: ( )에 해당하는 고유 키워드를 프롬프트에 포함하고 AI가 오답과 정답을 명확히 구분하는 문제를 생성한다.

---

## 문제 1 (빈칸 채우기 유형)

사용자가 한 번의 인증으로 여러 시스템이나 애플리케이션에 접근할 수 있게 해주는 인증 기술을 (        )이라고 한다.

**정답:** SSO(Single Sign-On)

---

## 문제 2 (직접 질문 유형)

사용자가 한 번의 로그인으로 여러 독립적인 소프트웨어 시스템에 재인증 없이 접근할 수 있게 하는 인증 메커니즘은 무엇인가?

**정답:** SSO(Single Sign-On, 싱글 사인온)

---

## 문제 3 (비교 설명 유형)

사용자가 기업 포털에 한 번 로그인하면 메일, 그룹웨어, ERP 등 다양한 업무 시스템에 별도 로그인 없이 접근할 수 있게 하는 통합 인증 기술은 무엇인가?

**정답:** SSO(Single Sign-On)
