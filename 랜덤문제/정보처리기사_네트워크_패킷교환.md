# 정보처리기사 네트워크 - 패킷교환

## AI 프롬프트 작성 가이드

- **정답 키워드 명시**: 프롬프트에 "정답: ( )"을 직접 지정하면 AI가 정답을 고정하여 문제를 역설계한다.
- **유형 지정**: 1.빈칸 채우기 / 직접 질문 / 비교 설명 유형을 명시하면 문제만 랜덤형식의 질문으로 다양성이 보장되어 작성한다. 2.정답은 ()으로 고정된다.
- **결정적 구분자 제시**: ( )에 해당하는 고유 키워드를 프롬프트에 포함하고 AI가 오답과 정답을 명확히 구분하는 문제를 생성한다.
- **해설 미포함**: 질문과 정답만 작성하며 해설은 작성하지 않는다.

---

## 문제 1

**유형**: 빈칸 채우기

데이터를 작은 단위인 패킷으로 분할하여 각각 독립적으로 전송하는 방식을 ( )이라고 한다.

**정답**: 패킷교환

---

## 문제 2

**유형**: 직접 질문

패킷교환 방식에서 데이터를 전송하기 전에 송수신 간에 물리적 경로를 미리 설정하는 방식을 무엇이라고 하는가?

**정답**: 가상회선

---

## 문제 3

**유형**: 비교 설명

회선교환과 달리 패킷교환에서는 통신 대역을 여러 사용자가 공유하며, 데이터가 독립적인 패킷 단위로 전송되므로 회선을 독점하지 않는 이점을 가지는데, 이처럼 통신 회선을 여러 사용자가 공유하는 방식을 무엇이라고 하는가?

**정답**: 다중화
