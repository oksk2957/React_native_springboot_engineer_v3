# 정보처리기사 소프트웨어분류 - 안티 스파이웨어(Anti-Spyware) 주관식 단답형 문제

> **정답 키워드:** 안티 스파이웨어 (Anti-Spyware)
> **문제 유형:** 주관식 단답형 (빈칸 채우기 + 직접 질문 + 비교 설명 혼합)

---

## 문제 1 (빈칸 채우기 유형)

**질문:** 보안 소프트웨어 유형 중, 사용자의 컴퓨터에 침투하여 사용자의 인터넷 사용 패턴, 키 입력, 개인정보 등을 몰래 수집하는 스파이웨어(Spyware)를 탐지하고 제거하는 기능을 수행하는 소프트웨어를 **[     ]** 라고 한다. 이 소프트웨어는 실시간 모니터링 기능을 제공하여 스파이웨어의 설치를 사전에 차단하거나, 이미 설치된 스파이웨어를 검색하여 제거한다.

**정답:** 안티 스파이웨어 (Anti-Spyware)

**개발 생각:** 빈칸 채우기 유형은 정보처리기사 실기 시험에서 가장 빈출되는 출제 형식이다. 문제 설명에 "스파이웨어 탐지·제거", "실시간 모니터링", "사용자 정보 몰래 수집 방지", "스파이웨어 설치 사전 차단"이라는 키워드가 포함되어 있으면, 정답은 안티 스파이웨어로 유도된다. 안티 스파이웨어의 어원인 'anti(반대)' + 'spyware(스파이웨어)'의 직관적 메타포가 결정적 구분자이다. 안티바이러스는 "바이러스·웜·트로이 목마 탐지·제거", 방화벽은 "네트워크 트래픽 필터링"이 키워드이므로, "스파이웨어 전용 탐지·제거"라는 표현이 안티 스파이웨어의 고유 구분자이다. 프롬프트 작성 시 이 키워드를 명시하면 AI가 정답을 안티 스파이웨어로 정확히 유도할 수 있다.

---

## 문제 2 (직접 질문 유형)

**질문:** 사용자의 동의 없이 인터넷 사용 패턴이나 개인정보를 몰래 수집하는 스파이웨어(Spyware)를 실시간으로 탐지하고 제거하는 기능을 수행하는 보안 소프트웨어의 이름은 무엇인가?

**정답:** 안티 스파이웨어 (Anti-Spyware)

**개발 생각:** 직접 질문 유형은 특정 보안 위협(스파이웨어)과 대응책(실시간 탐지·제거)을 연결하여 정답을 유도한다. "스파이웨어 실시간 탐지·제거", "개인정보 몰래 수집 방지"라는 구체적 기술 키워드가 안티 스파이웨어를 직접 지칭한다. 안티바이러스는 바이러스·웜 등이 핵심이고, 방화벽은 네트워크 접근 제어가 핵심이므로, "스파이웨어 전용 대응 소프트웨어"라는 보안 기능의 특성이 안티 스파이웨어 정답 유도에 효과적이다. 프롬프트에서 "스파이웨어 탐지·제거 전용 보안 소프트웨어"라고 지정하면 AI가 안티 스파이웨어를 정답으로 생성한다.

---

## 문제 3 (비교 설명 유형)

**질문:** 보안 소프트웨어 유형 중, **안티바이러스(Antivirus)**는 바이러스·웜·트로이 목마 등 악성코드를 탐지하고 제거하는 소프트웨어이고, **방화벽(Firewall)**은 네트워크 트래픽을 필터링하여 불법 접근을 차단하는 소프트웨어이다. 반면, **[     ]** 은(는) 사용자의 인터넷 사용 패턴, 키 입력, 개인정보 등을 몰래 수집하는 스파이웨어(Spyware)를 실시간으로 탐지하고 제거하는 기능을 수행하는 보안 소프트웨어이다. 이 용어의 어원은 'anti(반대)' + 'spyware(스파이웨어)'의 합성어이다.

**정답:** 안티 스파이웨어 (Anti-Spyware)

**개발 생각:** 비교 설명 유형은 정답 항목과 대조 항목(안티바이러스, 방화벽)을 동시에 제시하여, 대조 항목의 특성(바이러스·웜·트로이 목마 탐지·제거, 네트워크 트래픽 필터링)과 정답 항목의 특성(스파이웨어 실시간 탐지·제거, 개인정보 수집 방지)을 명확히 구분한다. 이 유형은 정보처리기사 시험에서 보안 소프트웨어 분류 용어들의 차이점을 묻는 변형으로 자주 출제된다. 특히 "어원에 'anti(반대)' + 'spyware(스파이웨어)'가 포함"이라는 메타포적 구분자를 추가하면, 안티 스파이웨어와 안티바이러스(Antivirus)의 기능적 차이를 명확히 할 수 있다. 프롬프트 작성 시 "안티바이러스·방화벽과 비교하여 [빈칸]의 특징을 설명하는 문제, 정답은 안티 스파이웨어, 어원 구분자 포함"이라고 지정하면, AI가 대조 항목 설명을 배치하고 빈칸에 안티 스파이웨어를 정답으로 유도하는 문제를 정확히 생성한다.

---

## AI 프롬프트 작성 가이드

안티 스파이웨어를 정답으로 하는 주관식 단답형 문제를 AI에게 생성하도록 요청할 때, 다음 프롬프트 형식을 사용한다:

```
정보처리기사 실기 시험용 주관식 단답형 문제 3개를 생성하라.
- 정답: 안티 스파이웨어 (Anti-Spyware)
- 유형: 빈칸 채우기와 직접 질문과 비교 설명을 혼합
- 각 문제에 정답과 개발 생각(해설)을 포함
- 안티 스파이웨어의 결정적 키워드(스파이웨어 탐지·제거, 실시간 모니터링, 개인정보 수집 방지, anti+spyware 어원)를 문제 설명에 배치하여 정답을 유도
- 안티바이러스, 방화벽 등 다른 보안 소프트웨어 분류 용어와의 비교 설명을 1개 문제에 포함
- 안티 스파이웨어와 안티바이러스의 기능적 유사성으로 인한 혼동 방지를 위해 어원 구분자(anti-spyware vs anti-virus)를 명시
```

**프롬프트 작성 핵심 원칙:**
1. **정답 키워드 명시**: 프롬프트에 "정답: 안티 스파이웨어"을 직접 지정하면 AI가 정답을 고정하여 문제를 역설계한다.
2. **유형 지정**: 빈칸 채우기 / 직접 질문 / 비교 설명 유형을 명시하면 문제 형식의 다양성이 보장된다.
3. **결정적 구분자 제시**: "스파이웨어 탐지·제거" vs "바이러스·웜 탐지(안티바이러스)" vs "네트워크 트래픽 필터링(방화벽)" 등 각 분류 용어의 고유 키워드를 프롬프트에 포함하면, AI가 오답과 정답을 명확히 구분하는 문제를 생성한다. 특히 안티 스파이웨어와 안티바이러스의 혼동 방지를 위해 어원(anti-spyware vs anti-virus)을 구분자로 활용한다.
4. **개발 생각 요청**: "개발 생각(해설)을 포함"이라고 지정하면, AI가 문제 설계 의도와 출제 패턴 분석을 함께 제공하여 학습 효율이 향상된다.
