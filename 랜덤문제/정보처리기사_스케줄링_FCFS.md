# 정보처리기사 스케줄링 - FCFS

## AI 프롬프트 작성 가이드

- **정답 키워드 명시**: 프롬프트에 "정답: ( )"을 직접 지정하면 AI가 정답을 고정하여 문제를 역설계한다.
- **유형 지정**: 1.빈칸 채우기 / 직접 질문 / 비교 설명 유형을 명시하면 문제만 랜덤형식의 질문으로 다양성이 보장되어 작성한다. 2.정답은 ()으로 고정된다.
- **결정적 구분자 제시**: ( )에 해당하는 고유 키워드를 프롬프트에 포함하고 AI가 오답과 정답을 명확히 구분하는 문제를 생성한다.
- **해설 미포함**: 질문과 정답만 작성하며 해설은 작성하지 않는다.

---

## 문제 1

**유형**: 빈칸 채우기

선착순으로 프로세스를 처리하는 비선점형 스케줄링 알고리즘을 ( )이라고 한다.

**정답**: FCFS

---

## 문제 2

**유형**: 직접 질문

FCFS 스케줄링에서 먼저 도착한 프로세스가 CPU를 먼저 할당받는 방식으로, 이로 인해 발생할 수 있는 대표적인 문제 현상은 무엇인가?

**정답**: 호위 현상(Convoy Effect)

---

## 문제 3

**유형**: 비교 설명

FCFS 스케줄링은 선착순으로 처리되므로 평균 대기 시간이 길어질 수 있다. 이 알고리즘의 영문 약어를 쓰시오.

**정답**: FCFS
