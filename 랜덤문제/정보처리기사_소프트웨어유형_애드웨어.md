# 정보처리기사 소프트웨어유형 - 애드웨어(Adware) 주관식 단답형 문제

> **정답 키워드:** 애드웨어 (Adware)
> **문제 유형:** 주관식 단답형 (빈칸 채우기 + 직접 질문 + 비교 설명 혼합)

---

## 문제 1 (빈칸 채우기 유형)

**질문:** 사용자가 소프트웨어를 무료로 사용할 수 있도록 대가로 **(     )** 를 반복적으로 표시하는 프로그램을 **(     )** 라고 한다. 이 방식은 소프트웨어 개발자가 수익을 창출하기 위해 사용자 인터페이스 내에 광고 배너나 팝업 광고를 삽입하여 노출하는 특징이 있으며, 과도한 광고 노출은 사용자 경험을 저하시킬 수 있다.

**정답:** 애드웨어 (Adware)

**개발 생각:** 빈칸 채우기 유형은 정보처리기사 실기 시험에서 가장 빈출되는 출제 형식이다. 문제 설명에 "무료 사용 대가", "광고 반복 표시", "광고 배너나 팝업"이라는 키워드가 포함되어 있으면, 정답은 애드웨어로 유도된다. 애드웨어의 어원인 'ad(광고)' + 'ware(소프트웨어)'의 직관적 메타포가 결정적 구분자이다. 프리웨어는 "무료 배포", 스파이웨어는 "사용자 정보 수집", 랜섬웨어는 "데이터 암호화·몸값 요구"가 핵심이므로, "광고 표시를 통한 수익 창출"이라는 표현이 애드웨어의 고유 구분자이다. 프롬프트 작성 시 이 키워드를 명시하면 AI가 정답을 애드웨어로 정확히 유도할 수 있다.

---

## 문제 2 (직접 질문 유형)

**질문:** 사용자가 특정 소프트웨어를 무료로 이용할 수 있게 하되, 대신 해당 프로그램 실행 중 지속적으로 광고 배너나 팝업 광고를 화면에 표시하여 개발자의 수익을 창출하는 방식의 소프트웨어 유형은 무엇인가?

**정답:** 애드웨어 (Adware)

**개발 생각:** 직접 질문 유형은 특정 비즈니스 모델(무료 사용 대가 광고 노출)과 기술적 특성(광고 배너·팝업 표시)을 연결하여 정답을 유도한다. "무료 이용", "광고 배너나 팝업 표시", "개발자 수익 창출"이라는 구체적 기술 키워드가 애드웨어를 직접 지칭한다. 프리웨어는 광고 없이 완전 무료가 핵심이고, 셰어웨어는试用 기간 후 구매 유도가 핵심이므로, "광고 노출을 통한 수익화"라는 비즈니스 모델이 애드웨어 정답 유도에 효과적이다. 프롬프트에서 "무료 사용 대가로 광고를 표시하여 수익을 창출하는 소프트웨어"라고 지정하면 AI가 애드웨어를 정답으로 생성한다.

---

## 문제 3 (비교 설명 유형)

**질문:** 소프트웨어 배포 모델 중 **프리웨어(Freeware)**는 사용자에게 완전 무료로 제공되며 광고가 없고, **셰어웨어(Shareware)**는 일정 기간 무료试用 후 정식 라이선스 구매를 유도한다. 반면, **[     ]** 은(는) 사용자가 소프트웨어를 무료로 사용할 수 있도록 대가로 광고를 반복적으로 표시하여 개발자가 수익을 창출하는 방식으로, '광고(advertisement)'의 약자가 어원에 포함된 소프트웨어 유형이다.

**정답:** 애드웨어 (Adware)

**개발 생각:** 비교 설명 유형은 정답 항목과 대조 항목(프리웨어, 셰어웨어)을 동시에 제시하여, 대조 항목의 특성(완전 무료·무광고,试用 후 구매 유도)과 정답 항목의 특성(무료 사용·광고 노출·수익 창출)을 명확히 구분한다. 이 유형은 정보처리기사 시험에서 소프트웨어 배포 모델들의 차이점을 묻는 변형으로 자주 출제된다. 특히 "어원에 '광고(ad)'가 포함"이라는 메타포적 구분자를 추가하면, 애드웨어와 프리웨어의 개념 혼동을 방지할 수 있다. 프롬프트 작성 시 "프리웨어·셰어웨어와 비교하여 [빈칸]의 특징을 설명하는 문제, 정답은 애드웨어, 어원 구분자 포함"이라고 지정하면, AI가 대조 항목 설명을 배치하고 빈칸에 애드웨어를 정답으로 유도하는 문제를 정확히 생성한다.

---

## AI 프롬프트 작성 가이드

애드웨어를 정답으로 하는 주관식 단답형 문제를 AI에게 생성하도록 요청할 때, 다음 프롬프트 형식을 사용한다:

```
정보처리기사 실기 시험용 주관식 단답형 문제 3개를 생성하라.
- 정답: 애드웨어 (Adware)
- 유형: 빈칸 채우기와 직접 질문과 비교 설명을 혼합
- 각 문제에 정답과 개발 생각(해설)을 포함
- 애드웨어의 결정적 키워드(광고 표시, 무료 사용 대가, 팝업·배너 광고, ad 어원)를 문제 설명에 배치하여 정답을 유도
- 프리웨어, 셰어웨어 등 다른 소프트웨어 배포 모델과의 비교 설명을 1개 문제에 포함
- 애드웨어와 프리웨어의 개념 유사성으로 인한 혼동 방지를 위해 어원 구분자(ad vs free)를 명시
```

**프롬프트 작성 핵심 원칙:**
1. **정답 키워드 명시**: 프롬프트에 "정답: 애드웨어"을 직접 지정하면 AI가 정답을 고정하여 문제를 역설계한다.
2. **유형 지정**: 빈칸 채우기 / 직접 질문 / 비교 설명 유형을 명시하면 문제 형식의 다양성이 보장된다.
3. **결정적 구분자 제시**: "광고 표시·무료 사용 대가" vs "완전 무료·무광고(프리웨어)" vs "试用 후 구매(셰어웨어)" 등 각 소프트웨어 배포 모델의 고유 키워드를 프롬프트에 포함하면, AI가 오답과 정답을 명확히 구분하는 문제를 생성한다. 특히 애드웨어와 프리웨어의 혼동 방지를 위해 어원(ad vs free)을 구분자로 활용한다.
4. **개발 생각 요청**: "개발 생각(해설)을 포함"이라고 지정하면, AI가 문제 설계 의도와 출제 패턴 분석을 함께 제공하여 학습 효율이 향상된다.
