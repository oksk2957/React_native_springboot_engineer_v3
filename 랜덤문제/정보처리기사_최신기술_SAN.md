# 정보처리기사 - 최신기술 (SAN)

## AI 프롬프트 작성 가이드

- **정답 키워드**: SAN (Storage Area Network)
- **유형**: 빈칸 채우기 / 직접 질문 / 비교 설명
- **결정적 구분자**: ( )에 해당하는 고유 키워드를 프롬프트에 포함하고 AI가 오답과 정답을 명확히 구분하는 문제 생성
- **작성 원칙**: 주관식 단답형, 정답은 ( )으로 고정, 해설 미작성

---

## 문제 1

서버와 저장장치 간에 전용 고속 네트워크를 구성하여 블록 수준의 데이터 접근을 제공하는 저장 기술은 무엇인가?

**정답:** (SAN)

---

## 문제 2

Fibre Channel 또는 iSCSI를 사용하여 대규모 데이터 센터에서 고성능 저장 자원을 여러 서버에 공유하는 전용 저장 네트워크는 무엇인가?

**정답:** (SAN)

## 문제 3

NAS와 달리 블록 단위로 데이터를 처리하며, 데이터베이스나 대규모 트랜잭션 처리에 적합한 고성능 저장 인프라는 무엇인가?

**정답:** (SAN)
