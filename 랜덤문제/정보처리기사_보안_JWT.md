# 정보처리기사 - 보안: JWT

## AI 프롬프트 작성 가이드

- **정답 키워드**: JWT (JSON Web Token)
- **유형**: 빈칸 채우기 / 직접 질문 / 비교 설명
- **결정적 구분자**: ( )에 해당하는 고유 키워드를 프롬프트에 포함하고 AI가 오답과 정답을 명확히 구분하는 문제 생성
- **작성 원칙**: 주관식 단답형, 정답은 ( )으로 고정, 해설 미작성

---

## 문제 1

JSON 형식으로 사용자의 인증 정보를 안전하게 전송하기 위해 사용되며, 헤더(Header), 페이로드(Payload), 서명(Signature)으로 구성되는 토큰 기반 인증 표준은 무엇인가?

**정답:** (JWT)

---

## 문제 2

클라이언트와 서버 간의 상태를 유지하지 않는(Stateless) 인증 방식에서 사용자의 신원을 검증하고 권한 정보를 담아 전달하는 JSON 기반 토큰 형식은 무엇인가?

**정답:** (JWT)

---

## 문제 3

Base64Url 인코딩을 사용하여 Header.Payload.Signature 형태로 표현되며, 주로 웹 애플리케이션의 인증(Authentication)과 인가(Authorization)에 활용되는 토큰 표준은 무엇인가?

**정답:** (JWT)
