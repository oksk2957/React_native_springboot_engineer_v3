# AI 프롬프트 작성 가이드

## 프롬프트 작성 핵심 원칙
1. **정답 키워드 명시**: 프롬프트에 "정답: ( )"을 직접 지정하면 AI가 정답을 고정하여 문제를 역설계한다.
2. **유형 지정**: 1.빈칸 채우기 / 직접 질문 / 비교 설명 유형을 명시하면 문제만 랜덤형식의 질문으로 다양성이 보장되어 작성한다. 정답은 ()으로 고정된다.
3. **결정적 구분자 제시**: ( )에 해당하는 고유 키워드를 프롬프트에 포함하고 AI가 오답과 정답을 명확히 구분하는 문제를 생성한다.

# 주관식 단답형 문제 - 워터링 홀 공격 (Watering Hole Attack)

## 문제 1
**유형**: 빈칸 채우기

특정 집단이 자주 방문하는 웹사이트를 미리 감염시켜 해당 집단의 사용자를 공격하는 방식을 ( )이라고 한다.

**정답**: (워터링 홀 공격)

---

## 문제 2
**유형**: 직접 질문

사냥꾼이 물웅덩이 주변을潛伏하며 사냥감을 기다리는 것처럼, 특정 타겟 집단이 자주 접속하는 웹사이트를 사전에 감염시켜 공격하는 기법의 이름은 무엇인가?

**정답**: (워터링 홀 공격)

---

## 문제 3
**유형**: 비교 설명

목표로 삼은 사용자들이 신뢰하는 특정 웹사이트를 미리 감염시켜 방문자들의 시스템에 악성코드를 전파하는 이 공격 기법은 ( )이며, APT 공격에서 자주 활용된다.

**정답**: (워터링 홀 공격)
