# 정보처리기사 주관식 단답형 문제 - 트러스트존 (Trust Zone)

## AI 프롬프트 작성 가이드
- **정답 키워드 명시**: 프롬프트에 "정답: ( )"을 직접 지정하면 AI가 정답을 고정하여 문제를 역설계한다.
- **유형 지정**: 1.빈칸 채우기 / 직접 질문 / 비교 설명 유형을 명시하면 문제만 랜덤형식의 질문으로 다양성이 보장되어 작성한다. 정답은 ()으로 고정된다.
- **결정적 구분자 제시**: ( )에 해당하는 고유 키워드를 프롬프트에 포함하고 AI가 오답과 정답을 명확히 구분하는 문제를 생성한다.

---

## 문제 1 (빈칸 채우기 유형)

ARM 프로세서에서 보안이 필요한 민감한 데이터를 처리하는 안전한 실행 환경을 (        )이라고 한다.

**정답:** 트러스트존

---

## 문제 2 (직접 질문 유형)

ARM 아키텍처에서 일반 애플리케이션이 실행되는 일반 실행 환경과 보안 애플리케이션이 실행되는 안전한 실행 환경을 분리하는 기술은 무엇인가?

**정답:** 트러스트존(Trust Zone)

---

## 문제 3 (비교 설명 유형)

일반 OS가 실행되는 일반 세계(Normal World)와 보안 OS가 실행되는 보안 세계(Secure World)로 하드웨어적으로 분리하여 민감한 데이터를 보호하는 ARM의 보안 기술은 무엇인가?

**정답:** 트러스트존(Trust Zone)
