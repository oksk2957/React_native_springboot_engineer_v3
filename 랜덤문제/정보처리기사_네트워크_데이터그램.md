# 정보처리기사 네트워크 - 데이터그램

## AI 프롬프트 작성 가이드

- **정답 키워드 명시**: 프롬프트에 "정답: ( )"을 직접 지정하면 AI가 정답을 고정하여 문제를 역설계한다.
- **유형 지정**: 1.빈칸 채우기 / 직접 질문 / 비교 설명 유형을 명시하면 문제만 랜덤형식의 질문으로 다양성이 보장되어 작성한다. 2.정답은 ()으로 고정된다.
- **결정적 구분자 제시**: ( )에 해당하는 고유 키워드를 프롬프트에 포함하고 AI가 오답과 정답을 명확히 구분하는 문제를 생성한다.
- **해설 미포함**: 질문과 정답만 작성하며 해설은 작성하지 않는다.

---

## 문제 1

**유형**: 빈칸 채우기

연결 설정 없이 독립적으로 전송되는 패킷 단위의 데이터를 ( )이라고 한다.

**정답**: 데이터그램

---

## 문제 2

**유형**: 직접 질문

데이터그램 방식에서 각 패킷이 독립적으로 전송되며 순서가 보장되지 않는 비연결형 통신을 제공하는 대표적인 인터넷 프로토콜은 무엇인가?

**정답**: UDP

---

## 문제 3

**유형**: 비교 설명

연결형 통신(TCP)은 데이터 전송 전에 연결을 설정하고 순서를 보장하는 반면, 데이터그램 방식은 연결 설정 없이 각 패킷을 독립적으로 전송한다. 이때 데이터그램 방식에서 패킷의 순서가 바뀌거나 중간에 손실될 수 있는 이유는 무엇인가?

**정답**: 비연결형
