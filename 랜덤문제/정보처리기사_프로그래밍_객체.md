# 정보처리기사 프로그래밍 - 객체

## AI 프롬프트 작성 가이드

- **정답 키워드 명시**: 프롬프트에 "정답: ( )"을 직접 지정하면 AI가 정답을 고정하여 문제를 역설계한다.
- **유형 지정**: 1.빈칸 채우기 / 직접 질문 / 비교 설명 유형을 명시하면 문제만 랜덤형식의 질문으로 다양성이 보장되어 작성한다. 2.정답은 ()으로 고정된다.
- **결정적 구분자 제시**: ( )에 해당하는 고유 키워드를 프롬프트에 포함하고 AI가 오답과 정답을 명확히 구분하는 문제를 생성한다.
- **해설 미포함**: 질문과 정답만 작성하며 해설은 작성하지 않는다.

---

## 문제 1

**유형**: 빈칸 채우기

데이터(속성)와 데이터를 처리하는 함수(메서드)를 하나로 묶은 소프트웨어 모듈을 ( )이라고 한다.

**정답**: 객체

---

## 문제 2

**유형**: 직접 질문

객체지향 프로그래밍에서 객체를 생성하기 위한 설계도 역할을 하며, 객체의 속성과 메서드를 정의하는 틀을 무엇이라 하는가?

**정답**: 클래스

---

## 문제 3

**유형**: 비교 설명

객체지향 프로그래밍의 핵심 개념 중 하나로, 객체의 내부 구현 세부사항을 외부에 숨기고 필요한 인터페이스만 노출하는 특성을 무엇이라 하는가?

**정답**: 캡슐화
