# AI 프롬프트 작성 가이드

## 프롬프트 작성 핵심 원칙
1. **정답 키워드 명시**: 프롬프트에 "정답: ( )"을 직접 지정하면 AI가 정답을 고정하여 문제를 역설계한다.
2. **유형 지정**: 1.빈칸 채우기 / 직접 질문 / 비교 설명 유형을 명시하면 문제만 랜덤형식의 질문으로 다양성이 보장되어 작성한다. 정답은 ()으로 고정된다.
3. **결정적 구분자 제시**: ( )에 해당하는 고유 키워드를 프롬프트에 포함하고 AI가 오답과 정답을 명확히 구분하는 문제를 생성한다.

---

# 주관식 단답형 문제 - 키로거 (Keylogger)

## 문제 1
**유형**: 빈칸 채우기

사용자의 키보드 입력을 몰래 기록하여 개인정보나 비밀번호 등을 탈취하는 악성 프로그램을 ( )라고 한다.

**정답**: (키로거)

---

## 문제 2
**유형**: 직접 질문

사용자가 키보드로 입력하는 모든 내용을 몰래 기록하여 공격자에게 전송하는 악성코드의 이름은 무엇인가?

**정답**: (키로거)

---

## 문제 3
**유형**: 비교 설명

사용자의 키보드 입력을 비밀리 모니터링하고 기록하여 계정 정보, 비밀번호, 신용카드 번호 등 민감한 데이터를 탈취하는 이 악성 소프트웨어는 ( )이며, 하드웨어 형태와 소프트웨어 형태로 존재한다.

**정답**: (키로거)
