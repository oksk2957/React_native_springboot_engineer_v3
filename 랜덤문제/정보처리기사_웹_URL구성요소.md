# 정보처리기사 웹 - URL 구성요소

## AI 프롬프트 작성 가이드

- **정답 키워드 명시**: 프롬프트에 "정답: ( )"을 직접 지정하면 AI가 정답을 고정하여 문제를 역설계한다.
- **유형 지정**: 1.빈칸 채우기 / 직접 질문 / 비교 설명 유형을 명시하면 문제만 랜덤형식의 질문으로 다양성이 보장되어 작성한다. 2.정답은 ()으로 고정된다.
- **결정적 구분자 제시**: ( )에 해당하는 고유 키워드를 프롬프트에 포함하고 AI가 오답과 정답을 명확히 구분하는 문제를 생성한다.
- **해설 미포함**: 질문과 정답만 작성하며 해설은 작성하지 않는다.

---

## 문제 1

**유형**: 빈칸 채우기

URL에서 웹 서버 내 리소스의 경로를 나타내며, 도메인 뒤에 오는 `/home/user`와 같은 부분을 ( )이라고 한다.

**정답**: path

---

## 문제 2

**유형**: 직접 질문

URL에서 `?` 뒤에 위치하며, `key=value` 형태로 데이터를 전달하는 부분을 무엇이라고 하는가?

**정답**: query

---

## 문제 3

**유형**: 비교 설명

URL에서 `#` 뒤에 위치하며, 웹 페이지 내 특정 섹션이나 앵커를 지정하여 브라우저가 해당 위치로 바로 이동하도록 하는 부분을 ( )이라고 한다.

**정답**: fragment
