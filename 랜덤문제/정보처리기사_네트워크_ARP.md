# 정보처리기사 주관식 단답형 문제 - ARP

## AI 프롬프트 작성 가이드

**정답 키워드:** ARP (Address Resolution Protocol)
**문제 유형:** 빈칸 채우기 / 직접 질문 / 비교 설명
**결정적 구분자:** ( ) 안에 들어갈 고유 키워드는 "ARP"이며, 오답과 정답을 명확히 구분하여 문제를 생성함.

---

## 문제 1 (빈칸 채우기)

네트워크 계층의 IP 주소를 데이터링크 계층의 물리적 주소(MAC 주소)로 변환하는 프로토콜은 ( )이다.

**정답:** ARP

---

## 문제 2 (직접 질문)

IP 주소에 대응하는 MAC 주소를 알아내기 위해 브로드캐스트로 요청 메시지를 보내고, 해당 호스트가 응답 메시지로 MAC 주소를 알려주는 프로토콜은 무엇인가?

**정답:** ARP

---

## 문제 3 (비교 설명)

다음 중 IP 주소를 MAC 주소로 매핑하는 프로토콜은 무엇인가?

**정답:** ARP
