# AI 프롬프트 작성 가이드

## 프롬프트 작성 핵심 원칙
1. **정답 키워드 명시**: 프롬프트에 "정답: ( )"을 직접 지정하면 AI가 정답을 고정하여 문제를 역설계한다.
2. **유형 지정**: 1.빈칸 채우기 / 직접 질문 / 비교 설명 유형을 명시하면 문제만 랜덤형식의 질문으로 다양성이 보장되어 작성한다. 정답은 ()으로 고정된다.
3. **결정적 구분자 제시**: ( )에 해당하는 고유 키워드를 프롬프트에 포함하고 AI가 오답과 정답을 명확히 구분하는 문제를 생성한다.

# 주관식 단답형 문제 - 타이포스쿼팅 (Typosquatting)

## 문제 1
**유형**: 빈칸 채우기

유명 웹사이트의 도메인 이름을 의도적으로 비슷하게 변형하여 사용자를 가짜 사이트로 유인하는 공격 기법을 ( )이라고 한다.

**정답**: (타이포스쿼팅)

---

## 문제 2
**유형**: 직접 질문

사용자가 "gooogle.com"처럼 실제 사이트와 유사한 철자의 가짜 도메인에 접속하도록 유도하여 피싱 공격을 수행하는 기법의 이름은 무엇인가?

**정답**: (타이포스쿼팅)

---

## 문제 3
**유형**: 비교 설명

도메인 이름의 철자를 살짝 바꾸거나 키보드 배열상 인접한 문자를 이용해 사용자를 속이는 이 공격은 ( )이며, 이는 사용자의 실수를 악용하는 대표적인 사회공학적 공격 기법 중 하나이다.

**정답**: (타이포스쿼팅)
