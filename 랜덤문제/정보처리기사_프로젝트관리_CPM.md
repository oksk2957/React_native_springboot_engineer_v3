# 정보처리기사 프로젝트 관리 - CPM

## AI 프롬프트 작성 가이드

- **정답 키워드 명시**: 프롬프트에 "정답: ( )"을 직접 지정하면 AI가 정답을 고정하여 문제를 역설계한다.
- **유형 지정**: 1.빈칸 채우기 / 직접 질문 / 비교 설명 유형을 명시하면 문제만 랜덤형식의 질문으로 다양성이 보장되어 작성한다. 2.정답은 ()으로 고정된다.
- **결정적 구분자 제시**: ( )에 해당하는 고유 키워드를 프롬프트에 포함하고 AI가 오답과 정답을 명확히 구분하는 문제를 생성한다.
- **해설 미포함**: 질문과 정답만 작성하며 해설은 작성하지 않는다.

---

## 문제 1

**유형**: 빈칸 채우기

프로젝트의 모든 작업을 노드와 화살표로 표현하고, 가장 긴 경로를 찾아 프로젝트 최소 완료 기간을 산정하는 기법을 ( )이라고 한다.

**정답**: CPM

---

## 문제 2

**유형**: 직접 질문

CPM에서 프로젝트 전체 기간에 영향을 주는 가장 긴 작업 경로로, 이 경로상의 작업이 지연되면 전체 프로젝트 기간이 지연되는 경로를 무엇이라고 하는가?

**정답**: 임계경로

---

## 문제 3

**유형**: 비교 설명

CPM에서 특정 작업의 시작이나 완료 시점을 지연시킬 수 있는 최대 시간으로, 해당 작업을 지연시켜도 전체 프로젝트 기간에 영향을 주지 않는 여유 시간을 무엇이라고 하는가?

**정답**: 여유시간
