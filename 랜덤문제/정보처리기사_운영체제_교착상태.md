# 정보처리기사 운영체제 - 교착상태

## AI 프롬프트 작성 가이드

- **정답 키워드 명시**: 프롬프트에 "정답: ( )"을 직접 지정하면 AI가 정답을 고정하여 문제를 역설계한다.
- **유형 지정**: 1.빈칸 채우기 / 직접 질문 / 비교 설명 유형을 명시하면 문제만 랜덤형식의 질문으로 다양성이 보장되어 작성한다. 2.정답은 ()으로 고정된다.
- **결정적 구분자 제시**: ( )에 해당하는 고유 키워드를 프롬프트에 포함하고 AI가 오답과 정답을 명확히 구분하는 문제를 생성한다.
- **해설 미포함**: 질문과 정답만 작성하며 해설은 작성하지 않는다.

---

## 문제 1

**유형**: 빈칸 채우기

두 개 이상의 프로세스가 서로의 자원을 기다리며 무한히 대기하는 상태를 ( )이라고 한다.

**정답**: 교착상태

---

## 문제 2

**유형**: 직접 질문

교착상태 발생의 4가지 필요 조건 중 하나로, 자원을 보유한 프로세스가 다른 자원을 요청할 때 보유 자원을 놓아주지 않는 조건은 무엇인가?

**정답**: 비선점

---

## 문제 3

**유형**: 비교 설명

교착상태를 예방, 회피, 탐지 및 복구하는 방법으로 해결할 수 있다. 교착상태를 방지하기 위해 은행원 알고리즘을 사용하는 방법을 무엇이라고 하는가?

**정답**: 교착상태 회피
