# 정보처리기사 주관식 단답형 문제 - UML (Unified Modeling Language)

## AI 프롬프트 작성 가이드
- **정답 키워드 명시**: 프롬프트에 "정답: ( )"을 짜절 지정하면 AI가 정답을 고정하여 문제를 역설계한다.
- **유형 지정**: 1.빈칸 채우기 / 직접 질문 / 비교 설명 유형을 명시하면 문제만 랜덤형식의 질문으로 다양성이 보장되어 작성한다. 정답은 ()으로 고정된다.
- **결정적 구분자 제시**: ( )에 해당하는 고유 키워드를 프롬프트에 포함하고 AI가 오답과 정답을 명확히 구분하는 문제를 생성한다.

---

## 문제 1 (빈칸 채우기 유형)

소프트웨어 시스템의 구조와 동작을 시각적으로 표현하기 위한 표준화된 모델링 언어를 (        )이라고 한다.

**정답:** UML(Unified Modeling Language)

---

## 문제 2 (직접 질문 유형)

객체지향 소프트웨어 개발에서 시스템의 구조적 측면(클래스 다이어그램, 컴포넌트 다이어그램 등)과 행위적 측면(유스케이스 다이어그램, 시퀀스 다이어그램 등)을 표현하는 표준화된 그래픽 언어는 무엇인가?

**정답:** UML(Unified Modeling Language, 통합 모델링 언어)

---

## 문제 3 (비교 설명 유형)

소프트웨어 개발 과정에서 개발자와 고객 간의 의사소통을 원활하게 하고, 시스템의 설계와 구조를 시각적으로 표현하는 데 사용되는 객체지향 표준 모델링 언어는 무엇인가?

**정답:** UML(Unified Modeling Language)
