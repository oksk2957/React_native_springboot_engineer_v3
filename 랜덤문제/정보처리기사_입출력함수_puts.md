# 정보처리기사 C언어 입출력함수 - puts() 주관식 단답형 문제

> **정답 키워드:** puts()
> **문제 유형:** 주관식 단답형 (빈칸 채우기 + 직접 질문 + 비교 설명 혼합)

---

## 문제 1 (빈칸 채우기 유형)

**질문:** C언어에서 문자열을 표준 출력(stdout)으로 출력할 때, 문자열 끝에 자동으로 줄바꿈(newline, '\n')을 추가하여 출력하는 함수는 **[     ]** 이다. 이 함수는 인자로 문자열 포인터를 받으며, 출력 성공 시 양수를 반환하고 실패 시 EOF를 반환한다.

**정답:** puts()

---

## 문제 2 (직접 질문 유형)

**질문:** C언어의 문자열 출력 함수 중, 출력 시 자동으로 줄바꿈 문자를 추가하므로 printf()에서 명시적으로 "\n"을 작성할 필요가 없는 함수는 무엇인가? 이 함수는 형식 지정자(%d, %s 등)를 지원하지 않고 문자열만 출력한다.

**정답:** puts()

---

## 문제 3 (빈칸 채우기 + 비교 설명 유형)

**질문:** C언어 문자열 출력 함수 중 printf()는 형식 지정자를 사용하여 다양한 데이터 타입을 출력하고 줄바꿈을 수동으로 지정해야 하지만, **[     ]** 함수는 형식 지정자 없이 문자열만 출력하며 줄바꿈을 자동으로 추가한다. 후자는 단순 문자열 출력에 더 효율적이지만, 정수나 실수 등 다른 타입은 출력할 수 없다.

**정답:** puts()

---

## AI 프롬프트 작성 가이드

puts()를 정답으로 하는 주관식 단답형 문제를 AI에게 생성하도록 요청할 때, 다음 프롬프트 형식을 사용한다:

```
정보처리기사 실기 시험용 주관식 단답형 문제 3개를 생성하라.
- 정답: puts()
- 유형: 빈칸 채우기와 직접 질문을 혼합
- puts()의 결정적 키워드(자동 줄바꿈, 형식 지정자 없음, 문자열만 출력, stdout)를 문제 설명에 배치하여 정답을 유도
- printf()와 비교하여 puts()의 차이점을 1개 문제에 포함
```

**프롬프트 작성 핵심 원칙:**
1. **정답 키워드 명시**: 프롬프트에 "정답: puts()"을 직접 지정하면 AI가 정답을 고정하여 문제를 역설계한다.
2. **유형 지정**: 빈칸 채우기 / 직접 질문 / 비교 설명 유형을 명시하면 문제 형식의 다양성이 보장된다.
3. **결정적 구분자 제시**: "자동 줄바꿈" vs "수동 줄바꿈" (printf), "형식 지정자 없음" vs "형식 지정자 지원" (printf) 등 각 함수의 고유 키워드를 프롬프트에 포함하면, AI가 오답과 정답을 명확히 구분하는 문제를 생성한다.
