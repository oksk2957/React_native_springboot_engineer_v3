## 주관식 단답형 문제 (사회공학 - 베이팅, 스피어피싱, 피싱, 파밍)

### 문제 1
**유형**: 빈칸 채우기
**질문**: 공공장소에 USB 등 저장매체를 의도적으로 버려두어, 이를 주운 사용자가 자신의 컴퓨터에 연결하도록 유도하여 악성코드를 감염시키는 사회공학 기법을 무엇이라고 하는가?
**정답**: (베이팅)(스피어피싱)(피싱)

---

### 문제 2
**유형**: 빈칸 채우기
**질문**: 특정 개인이나 조직을 목표로 삼아 맞춤형 공격을 수행하는 피싱 기법으로, 일반 피싱보다 치밀하고 정교한 공격 방식을 무엇이라고 하는가?
**정답**: (스피어피싱)(피싱)(파밍)

---

### 문제 3
**유형**: 빈칸 채우기
**질문**: 가짜 웹사이트를 만들어 사용자가 정상적인 사이트인 것처럼 속여 개인정보나 금융정보를 탈취하는 사이버 공격 기법을 무엇이라고 하는가?
**정답**: (파밍)(베이팅)(스피어피싱)

## AI 프롬프트 작성 가이드

### 프롬프트 예시

**문제 1 프롬프트:**
```
정보처리기사 시험 대비 주관식 단답형 문제를 작성해줘.
주제: 사회공학 공격 기법 중 베이팅(Baiting)
유형: 빈칸 채우기
정답: (베이팅)
요구사항:
- 정답 키워드 '베이팅'을 포함한 자연스러운 질문문장 작성
- 공공장소에 USB를 버려두고 사용자가 직접 연결하도록 유도하는 시나리오 포함
- 해설은 작성하지 않음
- 질문과 정답만 출력
```

**문제 2 프롬프트:**
```
정보처리기사 시험 대비 주관식 단답형 문제를 작성해줘.
주제: 사회공학 공격 기법 중 스피어피싱(Spear Phishing)
유형: 빈칸 채우기
정답: (스피어피싱)
요구사항:
- 정답 키워드 '스피어피싱'을 포함한 자연스러운 질문문장 작성
- 특정 개인이나 조직을 목표로 한 맞춤형 공격임을 강조
- 일반 피싱과의 차이점을 암시하는 문구 포함
- 해설은 작성하지 않음
- 질문과 정답만 출력
```

**문제 3 프롬프트:**
```
정보처리기사 시험 대비 주관식 단답형 문제를 작성해줘.
주제: 사회공학 공격 기법 중 파밍(Pharming)
유형: 빈칸 채우기
정답: (파밍)
요구사항:
- 정답 키워드 '파밍'을 포함한 자연스러운 질문문장 작성
- 가짜 웹사이트를 통한 정보 탈취 시나리오 포함
- 피싱과의 차이점(사이트 리다이렉트)을 암시하는 문구 포함
- 해설은 작성하지 않음
- 질문과 정답만 출력
```

### 프롬프트 작성 핵심 원칙
1. **정답 키워드 명시**: 프롬프트에 "정답: ( )"을 짜접 지정하면 AI가 정답을 고정하여 문제를 역설계한다.
2. **유형 지정**: 빈칸 채우기 / 직접 질문 / 비교 설명 유형을 명시하면 문제 형식의 다양성이 보장된다.
3. **결정적 구분자 제시**: ( )에 해당하는 고유 키워드를 프롬프트에 포함하고 AI가 오답과 정답을 명확히 구분하는 문제를 생성한다.
