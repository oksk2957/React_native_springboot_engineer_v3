# 정보처리기사 운영체제 - 임계구역

## AI 프롬프트 작성 가이드

- **정답 키워드 명시**: 프롬프트에 "정답: ( )"을 직접 지정하면 AI가 정답을 고정하여 문제를 역설계한다.
- **유형 지정**: 1.빈칸 채우기 / 직접 질문 / 비교 설명 유형을 명시하면 문제만 랜덤형식의 질문으로 다양성이 보장되어 작성한다. 2.정답은 ()으로 고정된다.
- **결정적 구분자 제시**: ( )에 해당하는 고유 키워드를 프롬프트에 포함하고 AI가 오답과 정답을 명확히 구분하는 문제를 생성한다.
- **해설 미포함**: 질문과 정답만 작성하며 해설은 작성하지 않는다.

---

## 문제 1

**유형**: 빈칸 채우기

여러 프로세스가 공유 자원에 접근할 때 데이터의 일관성을 위해 한 번에 하나의 프로세스만 접근할 수 있는 코드 영역을 ( )이라고 한다.

**정답**: 임계구역

---

## 문제 2

**유형**: 직접 질문

임계구역 문제를 해결하기 위한 조건 중 하나로, 두 개 이상의 프로세스가 동시에 임계구역에 들어가는 것을 막는 조건을 무엇이라고 하는가?

**정답**: 상호 배제

---

## 문제 3

**유형**: 비교 설명

임계구역에 진입하기 위해 프로세스가 사용하는 코드 구역(Entry Section), 임계구역(Critical Section), 임계구역에서 나오는 코드 구역(Exit Section)을 포함하는 전체 구조에서, 공유 자원에 실제로 접근하는 부분을 무엇이라고 하는가?

**정답**: 임계구역
