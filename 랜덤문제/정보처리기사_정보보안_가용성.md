# 정보처리기사 정보 보안 - 가용성

## AI 프롬프트 작성 가이드

- **정답 키워드 명시**: 프롬프트에 "정답: ( )"을 직접 지정하면 AI가 정답을 고정하여 문제를 역설계한다.
- **유형 지정**: 1.빈칸 채우기 / 직접 질문 / 비교 설명 유형을 명시하면 문제만 랜덤형식의 질문으로 다양성이 보장되어 작성한다. 2.정답은 ()으로 고정된다.
- **결정적 구분자 제시**: ( )에 해당하는 고유 키워드를 프롬프트에 포함하고 AI가 오답과 정답을 명확히 구분하는 문제를 생성한다.
- **해설 미포함**: 질문과 정답만 작성하며 해설은 작성하지 않는다.

---

## 문제 1

**유형**: 빈칸 채우기

인가된 사용자가 필요할 때마다 정보와 자원에 접근하고 사용할 수 있도록 보장하는 정보 보안의 3대 원칙 중 하나를 ( )이라고 한다.

**정답**: 가용성

---

## 문제 2

**유형**: 직접 질문

가용성을 보장하기 위해 시스템의 장애를 최소화하고, 장애 발생 시 빠르게 복구할 수 있도록 하는 정보 보안의 핵심 원칙은 무엇인가?

**정답**: 가용성

---

## 문제 3

**유형**: 비교 설명

가용성은 시스템이 필요할 때 항상 사용 가능하도록 하는 것이고, 기밀성은 인가된 사용자만 접근하도록 하는 것이다. 데이터가 불법적으로 변경되지 않도록 보장하는 정보 보안의 원칙은 무엇인가?

**정답**: 무결성
