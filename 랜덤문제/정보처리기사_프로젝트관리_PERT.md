# 정보처리기사 프로젝트 관리 - PERT

## AI 프롬프트 작성 가이드

- **정답 키워드 명시**: 프롬프트에 "정답: ( )"을 직접 지정하면 AI가 정답을 고정하여 문제를 역설계한다.
- **유형 지정**: 1.빈칸 채우기 / 직접 질문 / 비교 설명 유형을 명시하면 문제만 랜덤형식의 질문으로 다양성이 보장되어 작성한다. 2.정답은 ()으로 고정된다.
- **결정적 구분자 제시**: ( )에 해당하는 고유 키워드를 프롬프트에 포함하고 AI가 오답과 정답을 명확히 구분하는 문제를 생성한다.
- **해설 미포함**: 질문과 정답만 작성하며 해설은 작성하지 않는다.

---

## 문제 1

**유형**: 빈칸 채우기

작업 시간의 불확실성을 고려하여 낙관적, 비관적, 최다빈도 시간을 이용해 예상 시간을 산정하는 프로젝트 일정 관리 기법을 ( )이라고 한다.

**정답**: PERT

---

## 문제 2

**유형**: 직접 질문

PERT에서 작업 소요 시간을 추정할 때 사용하는 낙관적 시간, 최다빈도 시간, 비관적 시간의 세 가지 추정치를 활용하는 방법을 무엇이라고 하는가?

**정답**: 삼점 추정법

---

## 문제 3

**유형**: 비교 설명

PERT는 작업 시간의 불확실성을 고려하는 반면, 작업 시간을 확실하게 알고 있을 때 사용하는 결정적 일정 관리 기법은 무엇인가?

**정답**: CPM
