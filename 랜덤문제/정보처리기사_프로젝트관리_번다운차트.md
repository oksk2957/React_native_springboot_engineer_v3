# 정보처리기사 프로젝트 관리 - 번다운 차트

## AI 프롬프트 작성 가이드

- **정답 키워드 명시**: 프롬프트에 "정답: ( )"을 직접 지정하면 AI가 정답을 고정하여 문제를 역설계한다.
- **유형 지정**: 1.빈칸 채우기 / 직접 질문 / 비교 설명 유형을 명시하면 문제만 랜덤형식의 질문으로 다양성이 보장되어 작성한다. 2.정답은 ()으로 고정된다.
- **결정적 구분자 제시**: ( )에 해당하는 고유 키워드를 프롬프트에 포함하고 AI가 오답과 정답을 명확히 구분하는 문제를 생성한다.
- **해설 미포함**: 질문과 정답만 작성하며 해설은 작성하지 않는다.

---

## 문제 1

**유형**: 빈칸 채우기

스프린트나 반복 주기 동안 남아있는 작업량을 시각적으로 표현하여 프로젝트 진행 상황을 한눈에 파악할 수 있게 하는 그래프를 ( )이라고 한다.

**정답**: 번다운 차트

---

## 문제 2

**유형**: 직접 질문

번다운 차트에서 실제 작업 진행 속도를 나타내는 선이 이상적인 진행 속도선보다 위에 위치할 때, 이는 프로젝트가 어떤 상태임을 의미하는가?

**정답**: 지연 상태

---

## 문제 3

**유형**: 비교 설명

번다운 차트와 달리 시간에 따른 완료된 작업량을 누적하여 표현하는 차트로, 프로젝트의 전체 진행 추세를 보여주는 것은 무엇인가?

**정답**: 번업 차트
