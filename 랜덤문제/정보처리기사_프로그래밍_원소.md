# 정보처리기사 프로그래밍 - 원소

## AI 프롬프트 작성 가이드

- **정답 키워드 명시**: 프롬프트에 "정답: ( )"을 직접 지정하면 AI가 정답을 고정하여 문제를 역설계한다.
- **유형 지정**: 1.빈칸 채우기 / 직접 질문 / 비교 설명 유형을 명시하면 문제만 랜덤형식의 질문으로 다양성이 보장되어 작성한다. 2.정답은 ()으로 고정된다.
- **결정적 구분자 제시**: ( )에 해당하는 고유 키워드를 프롬프트에 포함하고 AI가 오답과 정답을 명확히 구분하는 문제를 생성한다.
- **해설 미포함**: 질문과 정답만 작성하며 해설은 작성하지 않는다.

---

## 문제 1

**유형**: 빈칸 채우기

집합이나 자료구조를 구성하는 개별적인 데이터 단위를 ( )이라고 한다.

**정답**: 원소

---

## 문제 2

**유형**: 직접 질문

배열이나 리스트 등의 자료구조에서 각 원소가 저장된 위치를 식별하는 데 사용하는 고유한 번호를 무엇이라 하는가?

**정답**: 인덱스

---

## 문제 3

**유형**: 비교 설명

배열은 연속적인 메모리 공간에 원소를 저장하여 인덱스 접근이 빠르지만, 연결 리스트는 각 원소가 다음 원소의 주소를 가리키는 방식으로 메모리를 효율적으로 사용한다. 연결 리스트에서 각 원소가 다음 원소를 가리키는 정보를 담고 있는 부분을 무엇이라 하는가?

**정답**: 포인터
