# 정보처리기사 웹 - AJAX

## AI 프롬프트 작성 가이드

- **정답 키워드 명시**: 프롬프트에 "정답: ( )"을 직접 지정하면 AI가 정답을 고정하여 문제를 역설계한다.
- **유형 지정**: 1.빈칸 채우기 / 직접 질문 / 비교 설명 유형을 명시하면 문제만 랜덤형식의 질문으로 다양성이 보장되어 작성한다. 2.정답은 ()으로 고정된다.
- **결정적 구분자 제시**: ( )에 해당하는 고유 키워드를 프롬프트에 포함하고 AI가 오답과 정답을 명확히 구분하는 문제를 생성한다.
- **해설 미포함**: 질문과 정답만 작성하며 해설은 작성하지 않는다.

---

## 문제 1

**유형**: 빈칸 채우기

웹 페이지를 전체 새로고침하지 않고도 서버와 비동기적으로 데이터를 교환하여 동적으로 업데이트하는 웹 개발 기술을 ( )라고 한다.

**정답**: AJAX

---

## 문제 2

**유형**: 직접 질문

AJAX에서 서버와 비동기 통신을 수행하는 핵심 객체로, JavaScript에서 HTTP 요청을 보내고 응답을 받는 데 사용되는 객체는 무엇인가?

**정답**: XMLHttpRequest

---

## 문제 3

**유형**: 비교 설명

기존 웹 방식은 페이지 전체를 새로고침하여 데이터를 갱신하는 반면, AJAX는 필요한 데이터만 비동기적으로 교환하여 사용자 경험을 향상시킨다. 이처럼 AJAX의 가장 큰 장점인 페이지 전체를 다시 로드하지 않고 부분적으로 데이터를 갱신하는 방식을 무엇이라 하는가?

**정답**: 비동기 통신
