# 정보처리기사 주관식 단답형 문제 - 정적 테스트

> - **정답 키워드 명시**: 프롬프트에 "정답: (정적 테스트)"을 직접 지정하면 AI가 정답을 고정하여 문제를 역설계합니다.

> - **작성 규칙**: 주관식 단답형, 해설 미작성, 질문과 정답만 기재, () 안에 단어 3개씩 묶어서 병렬 작성.
