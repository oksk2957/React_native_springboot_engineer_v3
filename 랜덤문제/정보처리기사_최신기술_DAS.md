# 정보처리기사 - 최신기술 (DAS)

## AI 프롬프트 작성 가이드

- **정답 키워드**: DAS (Direct Attached Storage)
- **유형**: 빈칸 채우기 / 직접 질문 / 비교 설명
- **결정적 구분자**: ( )에 해당하는 고유 키워드를 프롬프트에 포함하고 AI가 오답과 정답을 명확히 구분하는 문제 생성
- **작성 원칙**: 주관식 단답형, 정답은 ( )으로 고정, 해설 미작성

## 문제 1

서버에 직접 연결되어 사용하는 저장장치로, 서버 내부의 전용 케이블로 연결되며 확장성이 낮은 특징을 가진 저장 기술은 무엇인가?

**정답:** (DAS)

## 문제 2

**정답:** (DAS)

## 문제 3
