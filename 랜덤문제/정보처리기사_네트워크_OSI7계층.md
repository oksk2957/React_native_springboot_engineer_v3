# 정보처리기사 네트워크 - OSI 7계층

## AI 프롬프트 작성 가이드

- **정답 키워드 명시**: 프롬프트에 "정답: ( )"을 직접 지정하면 AI가 정답을 고정하여 문제를 역설계한다.
- **유형 지정**: 1.빈칸 채우기 / 직접 질문 / 비교 설명 유형을 명시하면 문제만 랜덤형식의 질문으로 다양성이 보장되어 작성한다. 2.정답은 ()으로 고정된다.
- **결정적 구분자 제시**: ( )에 해당하는 고유 키워드를 프롬프트에 포함하고 AI가 오답과 정답을 명확히 구분하는 문제를 생성한다.
- **해설 미포함**: 질문과 정답만 작성하며 해설은 작성하지 않는다.

---

## 문제 1

**유형**: 빈칸 채우기

국제표준화기구에서 제정한 네트워크 통신을 위한 7단계 참조 모델을 ( )이라고 한다.

**정답**: OSI 7계층

---

## 문제 2

**유형**: 직접 질문

OSI 7계층 중 물리적 연결을 담당하며 비트 단위의 데이터 전송을 수행하는 가장 낮은 계층은 무엇인가?

**정답**: 물리계층

---

## 문제 3

**유형**: 비교 설명

OSI 7계층 중 데이터의 암호화, 복호화, 압축 등을 수행하며 사용자 데이터의 문법적 표현을 담당하는 계층으로, 응용계층과 세션계층 사이에 위치하는 계층은 무엇인가?

**정답**: 표현계층
