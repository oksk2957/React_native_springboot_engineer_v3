# 정보처리기사 - 디자인패턴 (생성패턴)

## AI 프롬프트 작성 가이드

- **정답 키워드**: 생성패턴 (Creational Pattern)
- **유형**: 빈칸 채우기 / 직접 질문 / 비교 설명
- **결정적 구분자**: ( )에 해당하는 고유 키워드를 프롬프트에 포함하고 AI가 오답과 정답을 명확히 구분하는 문제 생성
- **작성 원칙**: 주관식 단답형, 정답은 ( )으로 고정, 해설 미작성

---

## 문제 1

객체의 생성 과정을 추상화하고 캡슐화하여, 객체를 생성하는 방법과 시점을 유연하게 제어하는 디자인 패턴 분류로 싱글톤, 팩토리 메서드, 추상 팩토리, 빌더, 프로토타입 등이 포함되는 것은 무엇인가?

**정답:** (생성패턴)
