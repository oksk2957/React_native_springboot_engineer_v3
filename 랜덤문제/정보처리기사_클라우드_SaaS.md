# 정보처리기사 클라우드 - SaaS

## AI 프롬프트 작성 가이드

- **정답 키워드 명시**: 프롬프트에 "정답: ( )"을 직접 지정하면 AI가 정답을 고정하여 문제를 역설계한다.
- **유형 지정**: 1.빈칸 채우기 / 직접 질문 / 비교 설명 유형을 명시하면 문제만 랜덤형식의 질문으로 다양성이 보장되어 작성한다. 2.정답은 ()으로 고정된다.
- **결정적 구분자 제시**: ( )에 해당하는 고유 키워드를 프롬프트에 포함하고 AI가 오답과 정답을 명확히 구분하는 문제를 생성한다.
- **해설 미포함**: 질문과 정답만 작성하며 해설은 작성하지 않는다.

---

## 문제 1

**유형**: 빈칸 채우기

클라우드 컴퓨팅 서비스 모델 중 하나로, 소프트웨어 애플리케이션을 인터넷을 통해 제공하는 방식을 ( )라고 한다.

**정답**: (SaaS)

---

## 문제 2

**유형**: 직접 질문

SaaS 환경에서 하나의 소프트웨어 인스턴스를 여러 고객(테넌트)이 공유하여 사용하는 아키텍처 패턴은 무엇인가?

**정답**: (멀티테넌시)

---

## 문제 3

**유형**: 비교 설명

SaaS는 IaaS(인프라), PaaS(플랫폼)와 함께 클라우드 서비스의 3대 모델로, 사용자가 소프트웨어를 직접 설치하거나 관리할 필요 없이 웹 브라우저를 통해 바로 사용할 수 있는 ( ) 형태의 서비스이다.

**정답**: (SaaS)
