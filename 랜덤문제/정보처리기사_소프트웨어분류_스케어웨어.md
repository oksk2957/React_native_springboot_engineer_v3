# 정보처리기사 소프트웨어분류 - 스케어웨어(Scareware) 주관식 단답형 문제

> **정답 키워드:** 스케어웨어 (Scareware)
> **문제 유형:** 주관식 단답형 (빈칸 채우기 + 직접 질문 + 비교 설명 혼합)

---

## 문제 1 (빈칸 채우기 유형)

**질문:** 악성 소프트웨어 유형 중, 사용자의 컴퓨터에 가상의 바이러스나 악성코드가 감염되었다는虚偽 경고 메시지를 반복적으로 표시하여, 사용자를驚嚇(scare)하고 불필요한 보안 소프트웨어 구매나 개인정보 유출을 유도하는 방식을 **[     ]** 라고 한다. 이 방식은 실제 악성코드가 없음에도 불안감을 조성하여 사용자를 속이는 사회공학적 공격 기법이다.

**정답:** 스케어웨어 (Scareware)

**개발 생각:** 빈칸 채우기 유형은 정보처리기사 실기 시험에서 가장 빈출되는 출제 형식이다. 문제 설명에 "虚偽 경고 메시지", "驚嚇(scare)", "불안감 조성", "불필요한 보안 소프트웨어 구매 유도"라는 키워드가 포함되어 있으면, 정답은 스케어웨어로 유도된다. 스케어웨어의 어원인 'scare(驚嚇하다)' + 'ware(소프트웨어)'의 직관적 메타포가 결정적 구분자이다. 랜섬웨어는 "데이터 암호화 후 몸값 요구", 스파이웨어는 "사용자 행동 감시 및 정보 수집", 트로이 목마는 "위장된 악성코드"가 키워드이므로, "虚偽 경고로 불안감 조성"이라는 표현이 스케어웨어의 고유 구분자이다. 프롬프트 작성 시 이 키워드를 명시하면 AI가 정답을 스케어웨어로 정확히 유도할 수 있다.

---

## 문제 2 (직접 질문 유형)

**질문:** 사용자에게 가짜 바이러스 감염 경고창을 지속적으로 노출시켜 실제로는 존재하지 않는 위협을 과장하여 보여주고, 이를 해결하기 위해 유료 보안 프로그램 구매를 유도하는 악성 소프트웨어의 이름은 무엇인가?

**정답:** 스케어웨어 (Scareware)

**개발 생각:** 직접 질문 유형은 특정 공격 시나리오(가짜 바이러스 경고창 노출)와 공격 목적(유료 보안 프로그램 구매 유도)을 연결하여 정답을 유도한다. "가짜 바이러스 감염 경고창", "실제로 존재하지 않는 위협 과장", "유료 보안 프로그램 구매 유도"라는 구체적 기술 키워드가 스케어웨어를 직접 지칭한다. 랜섬웨어는 실제 데이터 암호화가 핵심이고, 애드웨어는 광고 노출이 핵심이므로, "虚偽 경고로 제품 구매 유도"라는 공격 행위의 특성이 스케어웨어 정답 유도에 효과적이다. 프롬프트에서 "虚偽 바이러스 경고를 통한 유료 보안 소프트웨어 판매 유도 공격"이라고 지정하면 AI가 스케어웨어를 정답으로 생성한다.

---

## 문제 3 (비교 설명 유형)

**질문:** 악성 소프트웨어 유형 중, **랜섬웨어(Ransomware)**는 사용자의 파일을 암호화하고 복호화를 대가로 금전을 요구하고, **스파이웨어(Spyware)**는 사용자의 인터넷 사용 패턴이나 개인정보를 몰래 수집한다. 반면, **[     ]** 은(는) 실제 악성코드가 없음에도 불구하고 사용자의 컴퓨터에 바이러스가 감염되었다는虚偽 경고를 반복적으로 표시하여, 사용자를驚嚇하고 불필요한 보안 소프트웨어 구매를 유도하는 방식이다. 이 용어의 어원은 'scare(驚嚇하다)'에서 유래하였다.

**정답:** 스케어웨어 (Scareware)

**개발 생각:** 비교 설명 유형은 정답 항목과 대조 항목(랜섬웨어, 스파이웨어)을 동시에 제시하여, 대조 항목의 특성(데이터 암호화·금전 요구, 사용자 정보 몰래 수집)과 정답 항목의 특성(虚偽 경고·불안감 조성·제품 구매 유도)을 명확히 구분한다. 이 유형은 정보처리기사 시험에서 악성 소프트웨어 분류 용어들의 차이점을 묻는 변형으로 자주 출제된다. 특히 "어원에 'scare(驚嚇)'가 포함"이라는 메타포적 구분자를 추가하면, 스케어웨어와 스파이웨어(Spyware)의 발음 유사성으로 인한 혼동을 방지할 수 있다. 프롬프트 작성 시 "랜섬웨어·스파이웨어와 비교하여 [빈칸]의 특징을 설명하는 문제, 정답은 스케어웨어, 어원 구분자 포함"이라고 지정하면, AI가 대조 항목 설명을 배치하고 빈칸에 스케어웨어를 정답으로 유도하는 문제를 정확히 생성한다.

---

## AI 프롬프트 작성 가이드

스케어웨어를 정답으로 하는 주관식 단답형 문제를 AI에게 생성하도록 요청할 때, 다음 프롬프트 형식을 사용한다:

```
정보처리기사 실기 시험용 주관식 단답형 문제 3개를 생성하라.
- 정답: 스케어웨어 (Scareware)
- 유형: 빈칸 채우기와 직접 질문과 비교 설명을 혼합
- 각 문제에 정답과 개발 생각(해설)을 포함
- 스케어웨어의 결정적 키워드(虚偽 바이러스 경고, 불안감 조성, 보안 소프트웨어 구매 유도, scare 어원)를 문제 설명에 배치하여 정답을 유도
- 랜섬웨어, 스파이웨어 등 다른 악성 소프트웨어 분류 용어와의 비교 설명을 1개 문제에 포함
- 스케어웨어와 스파이웨어의 발음 유사성으로 인한 혼동 방지를 위해 어원 구분자(scare vs spy)를 명시
```

**프롬프트 작성 핵심 원칙:**
1. **정답 키워드 명시**: 프롬프트에 "정답: 스케어웨어"을 직접 지정하면 AI가 정답을 고정하여 문제를 역설계한다.
2. **유형 지정**: 빈칸 채우기 / 직접 질문 / 비교 설명 유형을 명시하면 문제 형식의 다양성이 보장된다.
3. **결정적 구분자 제시**: "虚偽 바이러스 경고" vs "데이터 암호화(랜섬웨어)" vs "사용자 정보 수집(스파이웨어)" 등 각 분류 용어의 고유 키워드를 프롬프트에 포함하면, AI가 오답과 정답을 명확히 구분하는 문제를 생성한다. 특히 스케어웨어와 스파이웨어의 혼동 방지를 위해 어원(scare vs spy)을 구분자로 활용한다.
4. **개발 생각 요청**: "개발 생각(해설)을 포함"이라고 지정하면, AI가 문제 설계 의도와 출제 패턴 분석을 함께 제공하여 학습 효율이 향상된다.
