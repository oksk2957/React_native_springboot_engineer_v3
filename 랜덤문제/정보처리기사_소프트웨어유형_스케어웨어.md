# 정보처리기사 소프트웨어유형 - 스케어웨어(Scareware) 주관식 단답형 문제

> **정답 키워드:** 스케어웨어 (Scareware)
> **문제 유형:** 주관식 단답형 (빈칸 채우기 + 직접 질문 + 비교 설명 혼합)

---

## 문제 1 (빈칸 채우기 유형)

**질문:** 악성 소프트웨어의 한 유형으로, 사용자의 컴퓨터가 심각한 보안 위험에 노출되었다는 **(     )** 메시지를 반복적으로 표시하여 공포심을 조성하고, 실제로는 불필요하거나 가짜인 보안 소프트웨어를 구매하도록 유도하는 프로그램을 **(     )** 라고 한다.

**정답:** 스케어웨어 (Scareware)

**개발 생각:** 빈칸 채우기 유형은 정보처리기사 실기 시험에서 가장 빈출되는 출제 형식이다. 문제 설명에 "공포심을 조성", "가짜 보안 소프트웨어 구매 유도", "불필요한 보안 프로그램"이라는 키워드가 포함되어 있으면, 정답은 스케어웨어로 유도된다. 스케어웨어의 어원인 'scare(겁주다)' + 'ware(소프트웨어)'의 직관적 메타포가 결정적 구분자이다. 랜섬웨어는 "데이터 암호화 후 몸값 요구", 스파이웨어는 "사용자 행동 감시 및 정보 수집", 애드웨어는 "광고 강제 표시"가 핵심이므로, "공포심 조성을 통한 가짜 소프트웨어 판매 유도"라는 표현이 스케어웨어의 고유 구분자이다. 프롬프트 작성 시 이 키워드를 명시하면 AI가 정답을 스케어웨어로 정확히 유도할 수 있다.

---

## 문제 2 (직접 질문 유형)

**질문:** 사용자에게 가짜 바이러스 감염 경고나 시스템 오류 메시지를 지속적으로 노출시켜 불안감과 공포심을 조성하고, 이를 해결하기 위해 유료 보안 프로그램이나 가짜 백신 소프트웨어의 구매를 유도하는 악성 프로그램의 이름은 무엇인가?

**정답:** 스케어웨어 (Scareware)

**개발 생각:** 직접 질문 유형은 특정 공격 패턴(심리적 압박)과 결과 행위(가짜 소프트웨어 구매 유도)를 연결하여 정답을 유도한다. "가짜 바이러스 경고", "불안감과 공포심 조성", "유료 보안 프로그램 구매 유도"라는 구체적 기술 키워드가 스케어웨어를 직접 지칭한다. 랜섬웨어는 실제 데이터 암호화가 핵심이고, 트로이 목마는 위장 배포가 핵심이므로, "심리적 공포를 이용한 가짜 제품 판매"라는 사회공학적 특성이 스케어웨어 정답 유도에 효과적이다. 프롬프트에서 "사용자 공포심을 조성하여 가짜 보안 소프트웨어를 판매하는 악성 프로그램"이라고 지정하면 AI가 스케어웨어를 정답으로 생성한다.

---

## 문제 3 (비교 설명 유형)

**질문:** 악성 소프트웨어 중 **랜섬웨어(Ransomware)**는 사용자 데이터를 암호화하고 해독을 대가로 금전을 요구하고, **스파이웨어(Spyware)**는 사용자의 행동을 몰래 감시하여 정보를 수집한다. 반면, **[     ]** 은(는) 사용자의 컴퓨터가 심각한 위험에 처했다는 가짜 경고를 반복적으로 표시하여 공포심을 조성하고, 실제로는 효과가 없거나 오히려 악성인 가짜 보안 소프트웨어를 판매하도록 유도하는 방식으로, '공포(scare)'라는 단어가 어원에 포함된 소프트웨어 유형이다.

**정답:** 스케어웨어 (Scareware)

**개발 생각:** 비교 설명 유형은 정답 항목과 대조 항목(랜섬웨어, 스파이웨어)을 동시에 제시하여, 대조 항목의 특성(데이터 암호화·몸값 요구, 행동 감시·정보 수집)과 정답 항목의 특성(공포심 조성·가짜 소프트웨어 판매 유도)을 명확히 구분한다. 이 유형은 정보처리기사 시험에서 악성 소프트웨어 유형들의 차이점을 묻는 변형으로 자주 출제된다. 특히 "어원에 '공포(scare)'가 포함"이라는 메타포적 구분자를 추가하면, 스케어웨어와 랜섬웨어(Ransomware)의 개념 혼동을 방지할 수 있다. 프롬프트 작성 시 "랜섬웨어·스파이웨어와 비교하여 [빈칸]의 특징을 설명하는 문제, 정답은 스케어웨어, 어원 구분자 포함"이라고 지정하면, AI가 대조 항목 설명을 배치하고 빈칸에 스케어웨어를 정답으로 유도하는 문제를 정확히 생성한다.

---

## AI 프롬프트 작성 가이드

스케어웨어를 정답으로 하는 주관식 단답형 문제를 AI에게 생성하도록 요청할 때, 다음 프롬프트 형식을 사용한다:

```
정보처리기사 실기 시험용 주관식 단답형 문제 3개를 생성하라.
- 정답: 스케어웨어 (Scareware)
- 유형: 빈칸 채우기와 직접 질문과 비교 설명을 혼합
- 각 문제에 정답과 개발 생각(해설)을 포함
- 스케어웨어의 결정적 키워드(공포심 조성, 가짜 보안 경고, 가짜 백신 판매 유도, scare 어원)를 문제 설명에 배치하여 정답을 유도
- 랜섬웨어, 스파이웨어 등 다른 악성 소프트웨어와의 비교 설명을 1개 문제에 포함
- 스케어웨어와 랜섬웨어의 개념 유사성으로 인한 혼동 방지를 위해 어원 구분자(scare vs ransom)를 명시
```

**프롬프트 작성 핵심 원칙:**
1. **정답 키워드 명시**: 프롬프트에 "정답: 스케어웨어"을 직접 지정하면 AI가 정답을 고정하여 문제를 역설계한다.
2. **유형 지정**: 빈칸 채우기 / 직접 질문 / 비교 설명 유형을 명시하면 문제 형식의 다양성이 보장된다.
3. **결정적 구분자 제시**: "공포심 조성·가짜 경고" vs "데이터 암호화·몸값 요구(랜섬웨어)" vs "행동 감시·정보 수집(스파이웨어)" 등 각 악성 소프트웨어의 고유 키워드를 프롬프트에 포함하면, AI가 오답과 정답을 명확히 구분하는 문제를 생성한다. 특히 스케어웨어와 랜섬웨어의 혼동 방지를 위해 어원(scare vs ransom)을 구분자로 활용한다.
4. **개발 생각 요청**: "개발 생각(해설)을 포함"이라고 지정하면, AI가 문제 설계 의도와 출제 패턴 분석을 함께 제공하여 학습 효율이 향상된다.
