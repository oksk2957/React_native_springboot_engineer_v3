# 정보처리기사 주관식 단답형 문제 - CPU 스케줄링

## AI 프롬프트 작성 가이드
- **정답 키워드 명시**: 프롬프트에 "정답: ( )"을 직접 지정하면 AI가 정답을 고정하여 문제를 역설계한다.
- **유형 지정**: 1.빈칸 채우기 / 직접 질문 / 비교 설명 유형을 명시하면 문제만 랜덤형식의 질문으로 다양성이 보장되어 작성한다. 정답은 ()으로 고정된다.
- **결정적 구분자 제시**: ( )에 해당하는 고유 키워드를 프롬프트에 포함하고 AI가 오답과 정답을 명확히 구분하는 문제를 생성한다.

---

## 문제 1 (빈칸 채우기 유형)

운영체제에서 CPU 사용권을 여러 프로세스에게 효율적으로 할당하기 위한 운영체제의 핵심 기법을 (        )이라고 한다.

**정답:** CPU 스케줄링

---

## 문제 2 (직접 질문 유형)

준비 상태(Ready Queue)에 있는 프로세스들 중에서 CPU를 할당받을 프로세스를 선택하고, CPU를 할당하는 시간을 결정하는 운영체제의 기법은 무엇인가?

**정답:** CPU 스케줄링

---

## 문제 3 (비교 설명 유형)

FCFS, SJF, Round Robin, Priority Scheduling 등의 알고리즘이 사용되는, 프로세스가 CPU를 사용하는 순서와 시간을 결정하여 시스템의 성능을 최적화하는 운영체제 기법은 무엇인가?

**정답:** CPU 스케줄링
