## 🎯 근본 오류 진단 (Root Cause Analysis)

Memory 전체 스캔(8개 폴더) + 실제 코드 대조를 통해 **4가지 근본 오류 카테고리**를 식별:

```
[A] Memory-코드 불일치 → 신경망 신뢰도 붕괴 → 다음 AI가 잘못된 판단
[B] 토큰 체인 단절 → data 0 rows → 앱 핵심 기능(통계/오답) 무용지물
[C] 미구현 TODO 잔존 → UX 블로커 → 사용자 경로 차단
[D] 코드 일관성 결여 → 잠재적 버그 → 유지보수 비용 증가
```

**핵심 통찰**: [A]가 다른 모든 오류를 악화시키는 메타오류. Memory가 현실을 반영하지 못하므로, 매 세션마다 잘못된 문제에 재투자.
