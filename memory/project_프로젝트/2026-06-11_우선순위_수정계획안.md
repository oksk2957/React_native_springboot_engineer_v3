### 수정 내용

#### Step 1: SQL 수정 (백엔드)
**파일**: `backend/src/main/resources/mapper/ProblemQueryMapper.xml:479-492`

**변경 전**:
```sql
SELECT id, 'FLASHCARD' AS card_type, question AS front_text, answer AS back_text,
       explanation, NULL AS option1, NULL AS option2, NULL AS option3, NULL AS option4, NULL AS option5,
       prog_language AS category, difficulty
FROM programming_language_problems
WHERE LOWER(prog_language) = LOWER(#{language})
ORDER BY id
```

**변경 후**:
```sql
SELECT id,
       CASE WHEN option1 IS NOT NULL AND option1 != '' THEN 'SUBJECTIVE' ELSE 'FLASHCARD' END AS card_type,
       question AS front_text, answer AS back_text, explanation,
       option1, option2, option3, option4, option5,
       prog_language AS category, difficulty
FROM programming_language_problems
WHERE LOWER(prog_language) = LOWER(#{language})
ORDER BY id
```

**전제 조건**: migration 005가 Supabase에 적용되어 `option1~5` 컬럼이 실제로 존재해야 함
