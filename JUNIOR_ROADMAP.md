## 📅 6개월 로드맵

| 기간 | 핵심 | 진행 |
|---|---|---|
| **1~2개월** | 언어 기초 + "왜?" | TypeScript/Java 기초, 각 코드에 "왜?" 답변 준비 |
| **3~4개월** | 응용 + 프로젝트 | 정보처리기사 앱 개발 (현 진행) |
| **5~6개월** | 고도화 + 취준 | 코드 리팩토링, 문서화, 이력서 |

## 🚀 현재 진행 (정보처리기사 앱)

| Phase | 내용 | 성장역량 |
|---|---|---|
| Phase 1 | Navigation + 화면 연결 | 프로젝트 구조 파악 |

| Phase 3 | 문제 풀이 로직 | 응용력 키우기 |
| Phase 4 | 데이터 저장 | 결과물 완성 |
| Phase 5 | 통계/차트 | 문서화能力 |
